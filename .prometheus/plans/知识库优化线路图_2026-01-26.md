**问题**: 当前需要手动运行 Dataview 查询检查健康度

**行动计划**:
- [ ] 创建自动化脚本（Python/Bash）定期运行健康检查
- [ ] 集成到 Templater 工作流
- [ ] 添加每日健康度快照到 0.DailyNotes/
- [ ] 设置健康度低于 75% 的预警机制

**技术方案**:
```python
# scripts/vault_health_monitor.py
- 检查孤立笔记（链接数 < 2）
- 检查索引覆盖率
- 检查标签规范性
- 生成每日健康报告
```

**预期收益**:
- 自动发现问题
- 减少手动维护工作量
- 保持知识库持续健康

### 优先级 P2: 中期优化（1-3 个月）

#### 4. **智能标签推荐系统**
**问题**: 手动添加标签工作量大，容易遗漏

**行动计划**:
- [ ] 分析现有标签使用模式
- [ ] 创建标签推荐规则库（基于目录、关键词）
- [ ] 开发自动化脚本：根据内容推荐标签
- [ ] 集成到笔记创建模板（Templater）
- [ ] 建立标签质量评分机制

**技术方案**:
```javascript
// .obsidian/plugins/templater/scripts/tag_recommender.js
- 基于文件所在目录推荐 Domain 标签
- 基于笔记内容推荐 Topic 标签
- 基于文件类型推荐 Type 标签
```

**预期收益**:
- 减少 50% 手动标签工作量
- 提升标签一致性
- 加速笔记创建流程
