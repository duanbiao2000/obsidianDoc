### CrewAI: Agent Matrix & Workflow

#### I. Agent Meta (核心定义)

- **Role**: 身份/职责 (ID)
- **Goal**: 任务/产出 (Outcome)
- **Backstory**: 经验/能力 (Proof)

---

#### II. Team: Dev Crew (LOL Model)

| Position    | Role (Title)   | Focus (核心)         |
| :---------- | :------------- | :----------------- |
| **Top**     | **PM**         | 战略 / 需求优先级         |
| **Jungle**  | **Tech Lead**  | 架构 / 技术排障          |
| **Mid**     | **Senior Dev** | 核心逻辑 / Code Review |
| **ADC**     | **Frontend**   | UI / 交互体验          |
| **Support** | **QA/DevOps**  | 质量 / 部署发布          |

---

#### III. Lifecycle (协作流)

1. **需求**: PM → PRD / 可行性报告
2. **架构**: Tech Lead → 技术方案 / 系统设计
3. **开发**: Senior & FE → 代码 / 单元测试
4. **质量**: QA Lead → 测试报告 / Bug List
5. **发布**: DevOps → 部署计划 / 发布日志

---

#### IV. Principles (核心原则)

- **唯一性**: 职责不重叠。
- **专家制**: 领域深度。
- **全覆盖**: 协同互补。
- **质量链**: 层层把控。
- **高反馈**: 信息透明。
