
### **From Messy Notes to Sharp Insights: 用 OKR Master 图形思考**

喂同学，今日我哋要 solve 嘅 problem 係：个脑有好多 idea，但係一盘散沙，点样将佢哋变成 structured, actionable insights？好多人睇完 [[图形思考]] 觉得好正，但係唔知点样落地。

答案就係用 OKR 呢个 a powerful framework，将 [[图形思考]] 嘅心法，变成你嘅 personal development roadmap。唔好再得个“知”字，我哋要“做到”。

---

#### **Phase 1: Set Your North Star — The Strategic Objective**

首先，要 set 一个够 high-level 嘅大目标。呢个就係你嘅 **Objective (O1)**。

*   **O1: 将图形思考内化为核心认知操作系统 (a core cognitive OS)，将混沌思绪 (messy thoughts) 转化为结构化、可行动的洞见 (structured, actionable insights)。**

**解读 (My two cents):**
你嘅终极目标唔係学识画几种图，而係要 upgrade 你个脑。就好似 iPhone 升级 iOS 咁，你嘅 an entire thinking process 都要 level up。呢个 Objective 够 aspirational，但又 a clear direction。

---

#### **Phase 2: The 3 Key Battles — Tactical Key Results (KRs)**

有咗 Objective，就要有 **Key Results (KRs)** 去量度你有冇 progress。呢三个 KRs 就係你今个 quarter 要打赢嘅三场仗。

*   **KR1: 用金字塔原理 KO 沟通黑洞**
    *   **Action:** 今季 `3` 个 core projects，`100%` 用金字塔原理去 structure 你嘅提案。
    *   **Metric:** 目标係将澄清 a logic 嘅沟通成本（例如来来回回开会、改 document）**降低 `50%`**。Very measurable, very solid。

*   **KR2: 用“深挖图”做福尔摩斯**
    *   **Action:** 针对 `2` 个 long-standing 嘅难题，用“深挖图”去搵 a root cause。
    *   **Metric:** 唔係净係画完就算，而係要 prove 你搵到嘅 a root cause 真係被解决咗。你要 deliver `2` 份 a clear analysis report and show the evidence。

*   **KR3: Build 你嘅 Personal Toolkit**
    *   **Action:** 跟住“守破离”原则，整返个 a personal graphic thinking toolkit，最少要有 `5` 种 core templates (金字塔、流程图 etc.)。
    *   **Metric:** 目标係解释 a complex concept 嘅时间**缩短 `30%`**。用图形，唔使讲咁多废话，a picture is worth a thousand words。

---

#### **Phase 3: Drill Down — The Execution Plan**

齋 set KRs 係唔够嘅，要有 a concrete execution plan。我哋用 **KR1 (金字塔原理)** 做例子，睇下点样 drill down。

*   **O2 (from KR1): 将金字塔原理 deep dive 到你嘅 a daily workflow**

    *   **KR2.1:** 喺 Obsidian 或者 Notion **整个 reusable template**，内置 self-check questions，确保你 a logic is solid。
    *   **KR2.2:** 喺 `5` 次 internal discussion **刻意练习**用口头金字塔结构，目标係收到 `3` 次 "your logic is very clear" 嘅 a positive feedback。
    *   **KR2.3:** 做完一个 a successful case，要**做个 post-mortem**，写篇文 share 出去，完成“守破离”里面“守”嘅 a feedback loop。

---

#### **Conclusion: From a Concept Tree to a State Machine**

So, you see? 呢套 OKR 唔係得个讲字。佢将 [[图形思考]] 呢个 a concept tree (理论)，变成一个 a state machine (可执行嘅步骤)。

你嘅 a top-level objective 直接回应笔记核心，而 every single KR 都係 a measurable, verifiable step。咁样先可以确保你唔係“学咗”，而係“学会咗”。

---

**一句话总结 (In a Nutshell):**

> 用 OKR 呢个 a powerful tool，将图形思考由一个 concept，变成 a set of measurable actions，将你嘅 a messy thought process，变成 an engine for creating insights。
