# [[个人发展-人生建议-速查手册]]：极简进化协议

## 核心公式
**系统胜率 = (盟友选择 $\times$ 内部操作系统) / 路径依赖** [^1]
- **盟友选择**：决定系统上限与下限。
- **内部操作系统**：专注力、紧迫感、可靠性。
- **价值杠杆**：从卖时间转向卖技能。

---

## 1. 决策架构矩阵

| 维度 | 核心逻辑 | 极简准则 |
| :--- | :--- | :--- |
| **关系层** | **协作杠杆** | 选错伴侣/盟友是系统最高成本错误 [^1]。做真实的自己以降低筛选成本。 |
| **心态层** | **内部动力** | 维持“一无所有”的饥饿感与极度的专注力 [^1]。 |
| **执行层** | **确定性** | 培养紧迫感；像一切取决于神一样祈祷，像一切取决于你一样工作 [^1]。 |
| **财富层** | **非线性增长** | 意识到百万不多是财富突变的锚点 [^1]；贩卖技能而非时间。 |

---

## 2. 失败模式识别 (Via Negativa)
通过 **减法 (Via Negativa)** 锁定系统下限，规避毁灭性路径 [^2]。

- **关系坍塌**：与总是赢的人打赌，或在能力圈外盲目坚持 [^2]。
- **认知套利失败**：笨人不停说（信息闭塞），聪明人不停问（模型采集） [^1]。
- **路径依赖陷阱**：无法解决当前工作问题却妄图创业；害怕损失导致错失杠杆 [^1]。
- **代谢过载**：缺乏平和的内心导致决策噪声过大。

---

## 3. 风险防御协议 (SOP)
1. **生存底线审计**：识别并切断“灭绝级”路径（如错误的关键关系） [^3]。
2. **任务递归分解**：将愿景压碎至无须思考即可执行的原子动作 [^4]。
3. **反常识检测**：拒绝绝对化思维，识别逻辑受前提约束的局限性 [^5]。
4. **快速放弃策略**：当价值预设与现实脱节时，通过“快速放弃”重建新锚点 [^3]。

---

## 4. 实践自检
- [ ] 我是在贩卖时间（线性），还是在打磨技能（杠杆）？
- [ ] 我的内部 OS 是否存在“拖延”或“分心”的底层漏洞？
- [ ] 关键盟友是系统的助力，还是负累？
- [ ] 面对困难时，我是在“梯度下降”逼近最优，还是在逃避？ [^6]
