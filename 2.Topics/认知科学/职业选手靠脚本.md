# 啟動腳本：`./boot_pro_os.sh`

職業選手不靠心情，靠腳本。
用 **Pro_OS** 強制覆蓋大腦預設的 **Resistance BIOS**。

---

### Phase 1: Pre-Boot (預加載)
*   **消除模糊：** 變量 `CURRENT_TASK` 必須精確。
    *   ❌ `export TASK="work on project"`
    *   ✅ `export TASK="refactor auth module in user-service.js"`
*   **環境淨化：** `kill -9` 所有非必要進程（瀏覽器 Tab、手機通知）。確保 `Context Switch` 成本為 0。

### Phase 2: Kernel Boot (引導)
*   **不可逆點：** 這是真理時刻。
*   **執行指令：** 不思考，不猶豫。
    *   `Step 1`: Open IDE.
    *   `Step 2`: Open File.
    *   `Step 3`: Start Typing.
*   **狀態：** 系統啟動成功。PID 已生成。

### Phase 3: Runtime (運行)
*   **信任調度：** OS 已接管，大腦只需響應當前線程。
*   **防火牆：** 所有硬體中斷（分心、情緒）皆被防火牆自動 `DROP`。拒絕上下文切換。

### Phase 4: Shutdown (優雅停機)
*   **海明威橋：** 停在最順手的地方，而非寫完的地方。
*   **配置寫入：** 導出環境變量 `NEXT_BOOT_TASK`。
*   **目的：** 確保明天的啟動摩擦力 (Boot Friction) 為 **Zero**。

---

**每天早上，腦內執行。`exit 0`。**
