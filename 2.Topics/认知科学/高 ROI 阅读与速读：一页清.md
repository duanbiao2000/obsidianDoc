# 高 ROI 阅读与速读：一页清

## 0. 核心公式

> 阅读 ROI ≈  
> **（主动回忆 × 针对难点练习 × 规划） /（被动重读 × 临时抱佛脚）**

目标不是「看得快」，而是：**记得住、用得出、考得好 / 做得好。**

---

## 1. 先避坑：5 个低 ROI 坏习惯 → 对应替代

| 坏习惯（避免） | 高 ROI 替代行为 |
|----------------|------------------|
| 被动反覆阅读，只扫文字 | 每一小节后：合上书，用 1–2 分钟**写/说出要点**（回想练习）。 |
| 只看解答，以为看懂=会做 | 先自己做 3–5 分钟，**卡住再看提示/解答**，然后闭书重做一遍。 |
| 考前疯狂突击 | 平时用**间隔+回想**，考前只做「总复盘」，不指望临时抱佛脚。 |
| 重复做已会的题，逃避难题 | 每次练习至少预留 30–50% 时间给**真正困难的题**。 |
| 学前不预览章节、习题 | 先看**目录 + 标题 + 小结 + 习题类型**，带着问题读。 |

---

## 2. 高 ROI 好方法：5 条可直接执行

1. **主动回忆（核心）**  
   - If 看完一小节，Then 合上书，用要点/思维导图/口头讲解的方式回忆。  

2. **难题先尝试，再切换（利用发散模式）**  
   - If 碰到难题，Then 先硬想 5 分钟 → 仍卡住就**标记后跳到其他题**，让潜意识继续处理。  

3. **每日计划，任务量化**  
   - 每天写下：今天要**完成的页数/题目/小结**，而不是「看一看这章」。  

4. **考试策略：由难而易跳跃法**  
   - 先快速扫一遍卷子：  
     - 秒会的先做完（保底分）  
     - 中等题标记，第二轮  
     - 最难的最后集中攻坚  

5. **相信「我可以变好」+ 坚持**  
   - 把「看不懂/不会」改为：「**现在还不会**，下一次多试几种方法。」  
   - 给自己设「**最少练到 X 分钟/几题再放弃**」的规则。

---

## 3. 速读：工具箱 vs 使用边界

### 3.1 速读方法 = 输入带宽工具（什么时候用）

- **结构速览（先抓全局）**
  - Previewing：目录、标题、小结、习题类型。
  - Skimming：快速看段首、段尾，抓主线。
- **精准定位（找信息）**
  - Scanning：为具体问题找特定数据/段落。
  - Meta-guiding：带着问题读（「这段在回答什么？」）。
- **提速与分块理解**
  - Chunking：按小节/主题为单位，而不是按句子。
  - Mind Mapping：一边读一边画知识结构图。
  - Z-Scan / RSVP：在**确认内容价值不高**时，用来快速筛选。

### 3.2 什么时候 **不能** 速读（必须放慢）

- 需要**深度理解与推理**：哲学、数学证明、复杂论文。
- 需要**品味/欣赏**：文学、诗歌、经典散文。
- 法律、合同、考试大纲等需要**精确理解细节**的文本。
- 为**长期记忆**而学的核心知识（专业基础课、关键概念）。

> 规则：  
> - **筛选、浏览、大量信息 → 可以速读**  
> - **要「懂透」「背住」「不出错」→ 慢读 + 回想 + 练习**

---

## 4. 一套通用「一章书」高 ROI 流程

1. **前 5 分钟：预览（速读模式）**  
   - 看目录、标题、小结、习题类型，写下：  
     - 这章在回答哪些问题？  
     - 有哪几块我已经熟？哪几块是全新概念？

2. **中段：精读 + 回想（深度模式）**
   - 读每小节时：  
     - 画简单结构图或列要点  
     - 读完立刻合上书，用自己的话写出这节的 3–5 个关键点
   - 碰到难题：先做 5 分钟 → 仍卡住就跳过，过一轮再回来。

3. **尾声 10–15 分钟：巩固与规划**
   - 用一页纸写：本章**3 个关键概念 + 2 道典型题解法**。
   - 标记「还不熟的点」，写进**明天/本周计划**中再练一次。
   - 视需要生成几道自己的检索题（问答/填空），供下次复习用。
