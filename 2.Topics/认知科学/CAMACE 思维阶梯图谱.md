# 🧠 CAMACE 思维阶梯图谱

> **核心心法**：思维是一场“爬塔”游戏。每上一层，不仅需要保留下一层的能力，更要完成一次认知维度的突变。

### 🗺️ 全景一览：从“复读机”到“造物主”

|    阶梯    | 层次                     | 核心动作 (动词)         | 爬塔隐喻 (具象化)                         | ⚠️ 关键边界 (不做糊涂虫)                            |
| :------: | :--------------------- | :---------------- | :--------------------------------- | :----------------------------------------- |
| **Lv.6** | **创造**<br>(Create)     | **重构/发明**<br>无中生有 | **造新路**<br>不走梯子了，用零件造出电梯或飞行器。      | **VS 评估**<br>评估是给现有打分，创造是抛开旧标准造新的。         |
| **Lv.5** | **评估**<br>(Evaluate)   | **判决/衡量**<br>价值判断 | **验梯子**<br>检查梯子结不结实，判断它是优质品还是次品。   | **VS 分析**<br>分析是客观拆解（是什么结构），评估是主观裁决（好不好用）。 |
| **Lv.4** | **分析**<br>(Analyze)    | **拆解/透视**<br>探究原理 | **拆梯子**<br>把梯子拆成木板和钉子，研究它为什么能承重。   | **VS 应用**<br>应用是“照着做”（求结果），分析是“拆开看”（求逻辑）。  |
| **Lv.3** | **应用**<br>(Apply)      | **执行/解决**<br>迁移使用 | **爬梯子**<br>拿起梯子解决“上房顶”的问题，不管梯子咋做的。 | **VS 理解**<br>理解停留在“脑子懂”，应用必须“动手做”。         |
| **Lv.2** | **理解**<br>(Understand) | **转化/复述**<br>消化吸收 | **进塔内**<br>走进塔里看见梯子，能跟别人描述梯子长啥样。   | **VS 记忆**<br>记忆是原样录音，理解是用自己的话转述。           |
| **Lv.1** | **记忆**<br>(Memory)     | **存储/回放**<br>复制粘贴 | **看塔名**<br>站在塔外死记硬背：“这是一座塔，叫X塔”。   | **VS 理解**<br>如果不明白意思只背字面，那就是记忆（如背古文）。      |

### 🔍 易混淆概念“除雾剂”

*(当你不确定自己处于哪一层时，用这些反例自查)*

#### 1. 它是【应用】还是【分析】？

- **场景**：使用勾股定理 $a^2+b^2=c^2$。
- **应用层**：套入数据 $3^2+4^2$，算出 $c=5$。 -> **关注解题结果**。
- **分析层**：推导公式来源，拆解为何是平方而不是立方。 -> **关注内部逻辑**。
- **一句话区分**：**“只会用工具”是应用，“能拆解工具”是分析。**

#### 2. 它是【评估】还是【创造】？

- **场景**：面对一个旧方案。
- **评估层**：列出优缺点，打分，说“这个方案不行”。 -> **基于现有标准的判断**。
- **创造层**：不管旧方案，提出一个前所未有的新路径。 -> **突破现有框架的生成**。
- **一句话区分**：**“当裁判”是评估，“做设计师”是创造。**

### 🚀 进阶心法：如何爬塔？

不要试图跳级。

- 没有**记忆**（素材），无法**理解**。
- 没有**理解**（内化），**应用**会出错。
- 不会**应用**（实践），**分析**就是纸上谈兵。
- 不懂**分析**（原理），**评估**就是瞎指挥。
- 无法**评估**（鉴赏），**创造**出的也是垃圾。

**学习的本质，就是在这六级台阶上，一步步把“死知识”变成“活智慧”。**
