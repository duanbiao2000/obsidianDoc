# 学习效果评估JOL：元认知校准协议

## 1. 核心模型
**学习效能：$Efficiency = \frac{实际掌握度}{预测掌握度 (JOL)} \times 校准频率$**

- **本质**：JOL (Judgment of Learning) 是对“我是否学会了”的预判。
- **陷阱**：**流畅性错觉**。看资料很顺滑 $\neq$ 大脑已建模。预测与现实的偏差越小，学习 ROI 越高。

## 2. JOL 评估执行矩阵

| 内容类型 | 评估时机 | 评估粒度 | 核心动作 |
| :--- | :--- | :--- | :--- |
| **概念性** (是什么) | 学习后延时 15min | 原子概念 | **再陈述**：不看资料能否用口语解释清楚？ |
| **程序性** (怎么做) | 练习过程中 | 关键步骤 | **判停检查**：脱离 SOP 能否独立完成闭环？ |
| **系统性** (为什么) | 跨章节复盘 | 逻辑连接 | **地图绘制**：能否画出概念间的因果链条？ |

## 3. 校准协议 (SOP)

### A. 信号审计 (Signal Check)
识别伪信号，防止 JOL 高估：
- **红灯 (伪信号)**：资料就在眼前、刚读完、听着很有道理。
- **绿灯 (真信号)**：能提取出关键词、能举出反例、能解释边界。

### B. 闭环校准流程
1. **预测**：在测试前，给自己当前的掌握程度打分 (0-100)。
2. **测试**：进行一次主动回忆 (Active Recall) 或实操测试。
3. **对比**：若 |预测 - 结果| > 20%，强制触发“重读模式”并记录错因。
4. **延迟评估**：24 小时后再次评估，消除短期记忆干扰。

## 4. 瞬时执行清单 (Checklist)

- [ ] **去错觉**：我现在的“学会感”是因为资料写得好，还是我记住了？
- [ ] **设延迟**：是否给 JOL 评估留出了至少 15 分钟的干扰间隔？
- [ ] **查边界**：我能否清晰说明这个知识点“不适用”于什么场景？
- [ ] **留存证**：我的预测分值与实际得分是否匹配？
