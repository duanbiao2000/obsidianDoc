## ✈️ [[无人机_飞控系统_角色]]

> 嵌入式系统是无人机的“大脑与神经系统”。

**核心函数关系表：**

| 模块   | 嵌入式系统作用             |
| ---- | ------------------- |
| 飞控   | 传感器读取、控制算法执行        |
| 导航   | 多源融合(EKF / RTK-GPS) |
| 任务执行 | 云台、相机、喷洒控制          |
| 通信   | 管理图传与遥控信号（MAVLink）  |
| 电源   | 电池监控与安全关机           |
| 安全机制 | 失联保护、地理围栏、紧急降落      |

**要点：**

- 嵌入式系统 = 中央处理器 + 实时操作系统 + 驱动层 + 控制算法。
- 无嵌入式系统，不可能实现智能、自主化飞行。

## ⚙️ [[无人机_嵌入式硬件_分类]]

> 无人机根据任务需求选择不同层次的嵌入式平台。

| 平台类型    | 应用场景       | 示例                          |
| ------- | ---------- | --------------------------- |
| MCU     | 小型无人机主控    | STM32F4/H7, NXP Kinetis     |
| 嵌入式SoC  | 图像处理/中高端机型 | Jetson, Qualcomm Flight RB5 |
| RTOS    | 实时性保障      | FreeRTOS, PX4/NuttX         |
| Linux系统 | 高算力任务      | Ubuntu on Jetson / Pi       |

**关键词：** PX4、ArduPilot、NuttX、Linux

## 🚀 [[嵌入式技术_推进无人机发展]]

> 嵌入式技术是无人机智能化的基础动力。

**作用方向：**

1. **自主能力提升**：AI芯片实现机载识别与避障
2. **可靠性增强**：看门狗、冗余传感器
3. **低功耗小型化**：SoC集成CPU/GPU/ISP
4. **复杂任务支持**：农业喷洒、物流投递

**趋势：** 嵌入式AI、异构计算、低功耗高可靠架构。

## 🌾 [[无人机_典型案例_农业植保]]

> 农业无人机是嵌入式控制的经典应用。

**系统结构：**

- RTK-GPS 实时定位
- IMU + 高度传感器：恒定高度控制
- PWM 驱动泵喷头：随速度调节流量
- MCU 实时执行控制逻辑（如 STM32）
