# 2020年代技术趋势 (2025更新版)

## ★ 数据架构演进

### Data Lakehouse成主流

```
数据仓库(结构化) + 数据湖(原始) 
            ↓ 融合
    Data Lakehouse (统一)
    - Delta Lake / Iceberg / Hudi
```

### Vector DB竞争格局

| 产品       | 定位   | 特点         |
| -------- | ---- | ---------- |
| Pinecone | 托管优先 | 易用，成本高     |
| Weaviate | 混合搜索 | 语义+关键词     |
| Chroma   | 轻量嵌入 | 开发调试友好     |
| Milvus   | 企业级  | 大规模分布式     |
| pgvector | 扩展型  | Postgres生态 |

- ★ 向量DB + 传统DB融合趋势
- △ 本地优先: SQLite + 向量扩展

```
          重要性↑
             │
  Prompt工程 ●────────● AI系统设计
             │
```

## △ 已过时/需降权的概念

```diff
- MLOps传统流程 → LLMOps/AI Engineering
- 从零训练模型 → 调用+微调基础模型  
- 特征工程为主 → Prompt工程为主
- 单一Vector DB → 混合检索架构
- DevOps边界清晰 → AI融入全流程
```

GOSIM 近年重点关注以下方向：

- **AI + 开源**：大模型基础设施、具身智能、智能体（Agent）互联网。
- **Rust 生态**：与 RustGlobal、RustChinaConf 联动，推动 Rust 在系统编程、AI 底层、浏览器引擎（如 Servo）等场景的应用。
- **开放协议与标准**：推动 Agentic Web 协议（如 ANP、MCP、A2A）、去中心化身份（DID/VC）、数据空间互操作性等。
- **跨芯片与跨平台**：支持国产算力、多模态模型部署、云边端协同。
