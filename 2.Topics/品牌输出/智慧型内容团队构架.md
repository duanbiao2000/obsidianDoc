## 1. 核心理念：专业驱动 + AI 杠杆
- **定位**: 聚焦财富管理、家族传承、全球资产配置（HNW 垂直领域）。
- **人格**: 信任度 + 情感温度。
- **底层**: 全流程 AI 工具链。

---

## 2. 角色矩阵 (Role Matrix)

| 角色 | 核心职责 | AI 协作方向 |
| :--- | :--- | :--- |
| **策划总监** | 选题、调性、审核 | 趋势扫描、选题建议 |
| **脚本创作者** | 深度文案、叙事结构 | 初稿生成、风格对标 |
| **视觉设计师** | 视觉体系、信息传达 | 图像生成、智能剪辑 |
| **运营经理** | 跨平台分发、转化路径 | 画像分析、互动脚本 |
| **数据分析师** | 指标体系、归因分析 | 自动化报表、增长建议 |
| **提示工程师** | 工具管理、模型微调 | Prompt 进化、质量监控 |

## 3. SOP：视频生产执行流 `production-pipeline`

| 节点 | 动作 | 负责人 | 关键输出 |
| :--- | :--- | :--- | :--- |
| **S1: 策划** | 构思主题、确定调性 | 策划 | 标记 01_引言 |
| **S2: 素材** | 导入图文/视听素材 | 编辑 | 标记 02_素材 |
| **S3: 粗剪** | 初步剪辑、删减冗余 | 剪辑 | 标记 03_段落 |
| **S4: 锚点** | `Shift+M` 创建章节标记 | 剪辑 | 标记 04_章节 |
| **S5: 视觉** | 添加章节标题/视觉锤 | 视觉 | 标记 05_标题 |
| **S6: 互动** | 植入跳转/问答模块 | 运营 | 标记 06_入口 |
| **S7: 语音** | ASR 识别 + 字幕微调 | AI/字幕员 | 标记 07_关键词 |
| **S8: 配音** | 录制/生成语音内容 | AI/配音 | 标记 08_语义 |
| **S9: 动画** | 添加动态转场 | 剪辑 | 标记 09_转场 |
| **S10: 特效** | 色彩校正、视觉增强 | 视觉 | 标记 10_高潮 |
| **S11: 审核** | 画面/文案合规性检查 | 主编 | 标记 11: 审稿 |
| **S12: 发布** | 导出、多平台排期 | 发布 | 标记 12: 上线 |
| **S13: 复盘** | 数据反馈、策略调整 | 数据 | 标记 13: 闭环 |

## 4. AI 工具栈 (The Stack)

- **文本**: Copilot, ChatGPT, DeepSeek.
- **视觉**: Midjourney, Runway, Canva AI.
- **知识管理**: Notion AI, Obsidian (原子化卡片).

---

## 5. 质量红线 (Checklist)

- [ ] **内容**: 调性是否契合 HNW 审美？知识密度是否达标？
- [ ] **视觉**: 信息传达效率（视觉锤）是否足够强？
- [ ] **运营**: 转化漏斗（预约咨询/课程报名）是否通畅？
- [ ] **效率**: AI 介入后的单位产出时间是否缩短 30% 以上？

---

## 6. 部署协议

1. **Phase 1 (验证)**: 主编 + 文案 + 视觉，打通内容生产闭环。
2. **Phase 2 (分发)**: 引入运营和跨平台调度工具。
3. **Phase 3 (优化)**: 引入数据分析和 Prompt 工程师，实现自动化增长。

**最后更新: 2026-01-01**

---

“HNW 审美”中的 **HNW** 是 **High Net Worth（高净值人士）** 的缩写，通常指可投资资产在100万美元以上（有时标准更高，如500万或3000万美元以上，视语境而定）的个人。这类人群往往具有较高的教育背景、国际视野、文化素养和消费能力。
