# Obsidian Spaced Repetition Plugin
## 卡片语法
### 单行卡片
以`::` 作为问题和答案之间的分隔符
### 多行卡
在问题与答案间插入空行, 并以 `??` 作为分隔符
### 填空卡
以`{{` 和 `}}`选中需要挖空内容
