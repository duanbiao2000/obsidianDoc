# 精英认知系统：终极操作手册

## 核心公式

**认知复利 = (行为改变 × 错误规避 × 决策压缩) ÷ 认知负荷**

精英不追踪"学了什么"，只追踪"改变了什么行为"和"避免了什么错误"。

---

## 第一部分：每日执行协议

### Stage 0：意图对齐（60秒）

| 动作 | 示例 |
|------|------|
| 今日主线问题 | "所有输入与行动服务于：推进XX决策" |
| 今日约束 | "只有2小时深度时间，只做1个Min-loop" |

### Stage 1：信息压缩（10分钟）

**输出格式：3-2-1**
- 3条高信号（事实/框架/反例）
- 2个问题（不确定点/可验证假设）
- 1个动作（48小时内可完成）

**信号评分（满分6分，低于4分丢弃）**

| 维度 | 判断标准 |
|------|----------|
| 可信度 | 证据/来源/可复核 |
| 杠杆度 | 能否影响决策或产出 |
| 可迁移性 | 能否跨场景复用 |

### 每日节奏（按能量分级）

**高能量日（30分钟）**
- 早间5min：主线+约束+Min-loop设定
- 日间15min：Min-loop执行
- 晚间10min：3-2-1压缩+预判偏差

**中能量日（15分钟）**
- 早间1min：主线问题
- 晚间14min：3-2-1压缩+1条If-Then

**低能量日（2分钟）**
- 只写一句："今天我注意到..."

## 第二部分：Min-loop设计规范

**必填5项**

| 项目 | 说明 |
|------|------|
| 触发器 | 什么时候发生 |
| 动作 | 最小动作（≤10分钟） |
| 对照 | 不做/旧做法是什么 |
| 指标 | 领先指标+滞后指标各1个 |
| 复跑次数 | 至少3次 |

**示例：职场沟通**
- 触发器：重要沟通前2分钟
- 动作：预判对方信息接收逻辑
- 对照：不预判直接沟通
- 领先指标：对方复述准确率
- 滞后指标：决策一次通过率

## 第三部分：周度校准（60分钟）

### Schema构建标准

必须是可调用形态（择一）：
- If-Then规则集
- 决策树
- Checklist
- 因果链图
- 错误分类器

**必填边界条件**："这个模型在什么情况下失效？"

### 认知差分（Diff）三问

1. 反例/失败模式是什么？
2. 如果我错了，最可能错在哪个前提？
3. 需要看到什么证据才会改变想法？

**输出格式**
- 命题："在XX场景下，A比B更有效"
- 前提："成立需要：条件1/2/3"
- 证伪标准："若出现X数据，则推翻"

### 周度必看4个数

| 指标 | 计算方式 |
|------|----------|
| 信号→行动转化率 | 高信号数 vs Min-loop数 |
| 闭环率 | 复跑≥3次 ÷ 总Min-loop |
| 预判命中率 | 预判正确 ÷ 总预判 |
| 负荷审计 | 哪些动作可持续vs让系统崩溃 |

## 第四部分：季度认知资产盘点（60分钟）

### 四库盘点

| 维度 | 盘点标准 | 精英实例 |
|------|----------|----------|
| 模型库 | 哪些框架被实际用于决策？ | 将SCQA升级为SCQA+利益锚点：沟通前强制写"对方KPI是什么" |
| 洞见库 | 哪些反常识结论经验证有效？ | "越重要的事越先给结论"→高管汇报决策时间缩短50% |
| 行动库 | 哪些微行动已无需意志力？ | 收到模糊请求时自动回复："你希望我解决的具体问题是什么？" |
| 盲区库 | 哪些领域暴露了无知？ | 低估非语言信号在跨文化谈判中的权重→导致合作失败 |

### 复利指标（可量化追踪）

| 指标 | 测量方式 | 意义 |
|------|----------|------|
| 相似问题处理速度 | 同类任务耗时对比（本季vs上季） | 时间释放=可投入更高杠杆事务 |
| 预判准确率趋势 | Min-loop中预判vs实际匹配度评分 | 减少救火消耗，提升心理带宽 |
| 决策后悔率趋势 | 每月末统计"不该做/该换做法"的决策数 | 后悔是认知税，越低系统越可靠 |

### 季度盘点时间分配

| 时间 | 动作 |
|------|------|
| 0-15min | 翻阅季度日志，只看失败信号和补丁记录 |
| 15-30min | 更新四库 |
| 30-45min | 计算三项复利指标，对比上季度 |
| 45-60min | 针对最大盲区设计下季度Min-loop |

## 第五部分：精英思维vs常人思维

| 常人做法 | 精英策略 |
|----------|----------|
| 盲目接收信息，陷入过载 | 逆向筛选：先问"不看会损失什么"，再决定是否输入 |
| 事必躬亲，拒绝外包 | 认知卸载：外包非核心事务，聚焦高杠杆工作 |
| 盲目社交，人脉无效 | 反社交杠杆：精准筛选，只维护可产生复利的关系 |
| 压力下情绪失控 | 延迟表达：情绪稳定≠不表达，而是选择时机表达 |
| 畏惧失败，不敢尝试 | 失败加速：主动设计低成本失败实验 |
| 墨守成规，照搬经验 | 环境适应：不同情境采用不同策略 |
| 追求阅读数量 | 只记录"哪本书改变了哪个行为" |
| 追求模型数量 | 宁可深度用好2个模型，不堆砌10个僵尸框架 |
| 美化盲区 | 盲区库必须包含"因无知导致的实际损失" |

## 第六部分：系统防崩机制

### 防断规则

| 机制 | 设计 |
|------|------|
| 最小存活动作 | 无论如何写1句"今天我注意到..." |
| 72小时重启 | 断掉超3天不追补，直接重新开始 |
| 产出护栏 | 连续3天只有输入无Min-loop→次日禁止新增输入 |

### 深潜触发条件

满足任一即触发：
- 同一问题连续3天出现
- 预判偏差连续3次同方向
- 认知债务超2周未处理
- 遭遇重大决策或挫折

### 深潜流程（60-90分钟）

1. 一句话定义核心卡点
2. 检索3-5个高质量信息源
3. 追问5次"为什么"或"如果不这样会怎样"
4. 更新/推翻原有Schema
5. 生成新的Min-loop
6. 设定7天后复盘提醒

**早间**

### 每月（30分钟）

- [ ] 执行率审计
- [ ] 价值审计：哪个动作ROI最高
- [ ] 负载审计：删除过重环节
- [ ] 认知债务清理

### 每季（60分钟）

- [ ] 四库盘点
- [ ] 三项复利指标计算
- [ ] 下季度实验设计

## 核心护栏

**每天至少1个动作必须直接提升现实交付**

沟通更快、文档更清晰、决策更稳、返工更少

**终极检验**

> 如果明天状态极差、时间极少，你仍然会做的那1个动作是什么？

那就是你真正的核心。

---

## 精英认知复利的本质

不是积累信息，而是压缩决策路径。

每一次盘点都在回答：
- "我少做了什么蠢事？"（后悔率↓）
- "我快做了什么对事？"（处理速度↑）
- "我准做了什么预判？"（准确率↑）

> 能跑10年的70分系统，胜过跑3个月就放弃的100分系统。
