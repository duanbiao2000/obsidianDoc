## Awesome GitHub Copilot 贡献指南总结  
  
### 贡献方式  
- **Instructions**: 自定义 GitHub Copilot 行为的指令文件，存放于 `instructions/` 目录  
- **Prompts**: 预设的开发场景模板，存放于 `prompts/` 目录  
- **Chat Modes**: 专业领域的聊天模式配置，存放于 `chatmodes/` 目录  
- **Collections**: 相关内容的集合，便于发现和使用完整工具包，存放于 `collections/` 目录  
  
### 文件格式要求  
- 使用小写文件名，以连字符分隔  
- 每种类型有特定的文件扩展名：  
- `*.instructions.md` (指令)  
- `*.prompt.md` (提示)  
- `*.chatmode.md` (聊天模式)  
- `*.collection.yml` (集合)
