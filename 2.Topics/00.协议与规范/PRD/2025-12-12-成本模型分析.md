## 2. 核心单位成本 (Unit Economics)

### A. 技术栈单价 (Gemini Flash + 缓存方案)

- **LLM**: `$0.07 / 轮` (已计入 90% 缓存折扣)。
- **ASR (语音识别)**: `$0.15 / 对话` (8分钟 Advanced 识别)。
- **TTS (语音合成)**: `$0.004 / 对话` (50% 开启率)。
- **Infra (计算)**: `$0` (Cloud Run 免费额度全覆盖)。

### B. 综合函数

- **单用户月成本 (DAU)**: `~$0.22 / 月`。
- **单注册用户月收入 (ARPU)**: `$0.055 / 月` (2.8% 转化率)。

## 4. 生存优化协议 `optimization-steps`

### 第一阶段：技术降本 (必须执行)

1. **上下文复用**: 强制 1 小时内 Session 缓存，轮 2-10 成本削减 90%。
2. **混合推理**:
   - 简单对话 -> `Mistral 7B` (本地/廉价 API)。
   - 语法纠正 -> `Gemini Flash` (高价值调用)。
3. **日志脱水**: 从全量日志转向 Error-Only，运维成本压缩 50%。

### 第二阶段：商业提效 (必须执行)

1. **ARPU 提振**: 引入 $15+ 高端订阅 (自定义 AI 角色 + 离线模式)。
2. **转化锚点**: 免费体验从 3 次增至 7 次，目标转化率从 1% 提升至 3%。

---

## 5. 战略转向：B2B2C 路径

**B2C 经济模型已破产**。必须转向机构付费：

- **目标**: 学校/培训机构。
- **定价**: `$20 / 学生 / 年` (机构批量采购)。
- **逻辑**: 利用机构预算抵消 LLM 高昂的边际成本，实现基础流量 Break-even。
