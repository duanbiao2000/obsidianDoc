# 早晨第一件事 v3.0

## 本质

```
早晨 ≠ 完成任务
早晨 = 给复利引擎加注燃料
```

---

## 框架：2×2 价值矩阵

|  | **内在** | **外在** |
|---|---|---|
| **输入** | ① 内核升级 | ③ 市场扫描 |
| **输出** | ② 系统重构 | ④ 价值交付 |

---

## 四象限

**① 内核升级** → 提升思考深度
- 深度阅读 → 提炼模型
- 复盘失败 → 抽象原则
- 无主题思考 → 探索盲区

**② 系统重构** → 优化执行效率
- 迭代工作流
- 对齐 OKR
- 设计高价值习惯

**③ 市场扫描** → 发现外部机会
- 研究趋势
- 分析需求
- 信息访谈

**④ 价值交付** → 构建可定价资产
- 深度写作
- 开源贡献
- 作品集更新

---

## 依赖关系

```
① → ② → ③ → ④
认知 → 能力 → 机会 → 交付
```

> 跳过前置步骤直接输出 = 不可持续

---

## 类比

```
早晨 = 编译时间 (compile-time)
白天 = 运行时间 (run-time)
```
