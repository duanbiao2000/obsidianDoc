# 平台禁书：认知边界与获取协议

## 1. 核心逻辑：书籍可见度 = 系统容忍度
书籍在电商平台的可见度是观察**系统审查边界**的温度计。
- **本质**：禁书推荐旨在激发**认知独立**与**权力批判**。
- **现状**：涉及极权批判、去中心化、非官方宗教的书籍在大陆主流电商（京东/当当/淘宝）处于常态化下架或受限状态。

## 3. 获取协议：突破信息壁垒 (Actionable)

1.  **物理跨界 (Import)**：通过淘宝全球购、YesAsia 或前往港台购买繁体版（如远流、联经出版）。
2.  **数字影子 (Digital)**：利用 **Z-Library** 或 **Library Genesis** 检索电子版（需注意网络安全）。
3.  **学术途径 (Archive)**：关注高校图书馆或外文书店的学术专柜，寻找原版书。
4.  **按需即读 (JIT)**：对此类书籍，建议“发现即购买”，规避突发性全网下架风险。

## 4. 风险警示 (Safeguards)
- **IF** 涉及公开传播或大批量分发禁书 $\rightarrow$ **DANGER** 可能触及法律/社交红线。
- **THINK** 独立阅读，低调交流，保护认知带宽。
