# [[语言能力日练计划]]：高强度提分协议

## 1. 核心算式
**习得效能 = (主动输出 $\div$ 被动输入) $\times$ 反馈闭环**

| 维度 | 核心行动 (SOP) | 交付物 | 价值 (ROI) |
| :--- | :--- | :--- | :--- |
| **听 (Listen)** | **精听 Shadowing**：跟读 3min 播客，模仿弱读/连读。 | 改错记录 | 建立母语级听力反射 |
| **说 (Speak)** | **录音复述**：针对今日输入进行 2min 总结并录音。 | 自评录音 | 纠正发音代偿与语法冗余 |
| **读 (Read)** | **结构化速读**：300 词短文，抓取逻辑链与核心论点。 | 逻辑脑图 | 训练高层级信息提取能力 |
| **写 (Write)** | **语义转译**：将母语摘要转译为目标语，追求地道搭配。 | 转译文档 | 消除“中式思维”逻辑阻碍 |
| **基 (Base)** | **Anki 增量复习**：生词 $\rightarrow$ 例句 $\rightarrow$ 场景应用。 | 记忆库更新 | 固化核心表达词库 |

## 2. 每日执行协议 (30-60min)

### A. 启动阶段 (10min)
- **Anki 扫荡**：复习昨日生词。
- **影子跟读**：选一段 1min 音频，强制同步语音语调。

### B. 核心攻坚 (30min)
- **输入转输出**：读/听一份材料 $\rightarrow$ **强制生成**：写 100 词摘要或讲 1min 复述。
- **查缺补漏**：查找复述过程中“卡壳”的表达，记入 Anki。

### C. 反馈修正 (10min)
- **回听对比**：回听录音，对照原文识别漏听或发音偏差。
- **AI 润色**：将写作片段交给 AI，分析地道表达与语法错误。

## 3. 难度调节法则 (Zone of Proximal Development)
- **i+1 准则**：材料难度应处于“略微吃力但能懂 80%”区间。
- **判停阈值**：若某模块错误率 < 5% $\rightarrow$ **强制提速/增难**。
- **短板聚焦**：若连续 3 天 RPE（主观疲劳度）过低，说明强度不足，需增加输出权重。

## 4. 瞬时执行清单 (Checklist)
- [ ] **质量检查**：今日是否完成了“录音+回听”闭环？
- [ ] **输出比例**：输出时间是否占总时长的 ≥ 50%？
- [ ] **知识入库**：是否有至少 3 个新地道表达进入 Anki？
- [ ] **审计**：是否在真实/模拟场景中刻意使用了新词汇？
