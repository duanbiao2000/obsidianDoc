# 高效编程键盘协议：肌肉记忆优化指南

## 1. 核心原则

**效率逻辑：$Efficiency = \frac{1}{手指位移距离 \times 组合键复杂度}$**

- **目标**：手掌保持在主键区（Home Row），消除寻找边缘键（Esc/方向键）的位移。

## 2. 键位重映射矩阵 (Remap Matrix)

| 原始键             | 目标映射           | 场景/价值                   | 操作频率 |
| :-------------- | :------------- | :---------------------- | :--- |
| **Caps Lock**   | **Esc / Ctrl** | Vim 模式切换 / 快捷键核心        | 极高   |
| **Ctrl + [**    | **Esc**        | 原生支持，无需配置 (Vim/VS Code) | 高    |
| **Ctrl + hjkl** | **方向键**        | 全局光标移动（Vim 风格）          | 极高   |
| **Alt + h/l**   | **跳单词**        | 快速跨单词移动                 | 高    |
| **Caps + BS**   | **删除整行**       | 快速清理代码行                 | 中    |
| **右 Alt + [**   | **{**          | 消除 Shift 组合，直接输入符号      | 中    |

## 3. 系统级操作协议 (SOP)

### A. 窗口与标签管理

- **应用切换**：`Alt + Tab` (Win) / `Cmd + Tab` (Mac)。
- **标签切换**：统一映射为 `Ctrl + Tab` (正向) / `Ctrl + Shift + Tab` (反向)。
- **快速唤起**：设置 `Alt + Space` 或 `Option + Space` 唤起终端/Raycast/Alfred。

### B. 编辑器深度优化 (IDE)

- **跳转定义**：映射至 `Alt + .` 或鼠标侧键。
- **代码格式化**：映射至 `Ctrl + ;`（单手可触）。
- **重构/重命名**：映射至 `Alt + R`。

## 4. 推荐工具栈 (Tool Stack)

| 平台          | 核心工具                    | 核心功能             |
| :---------- | :---------------------- | :--------------- |
| **macOS**   | **Karabiner-Elements**  | 复杂键位改写 (JSON 配置) |
| **Windows** | **AutoHotkey (AHK)**    | 脚本化定制任何快捷键       |
| **Linux**   | **xmodmap / i3wm**      | 键位映射与平铺窗口管理      |
| **跨平台**     | **Raycast / PowerToys** | 全局搜索与窗口快速排版      |

## 5. 瞬时执行清单 (Checklist)

- [ ] **核心改写**：Caps Lock 是否已改为 Esc/Ctrl？
- [ ] **导航优化**：是否能不移动右手手掌就操作光标（hjkl）？
- [ ] **符号提速**：高频符号（{}[]）是否实现了单键或低复杂度组合？
- [ ] **肌肉记忆**：每日选择一个新映射进行 10 分钟刻意练习。
