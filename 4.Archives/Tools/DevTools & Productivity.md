# 开发者工具栈：精简指南 (ByteByteGo)

**原则：人生苦短，善用工具 $\rightarrow$ 极致效率。**

---

### 1. 开发环境 (Env)

- **VS Code**：全能、插件库。
- **IntelliJ / PyCharm**：JetBrains 系，强重构、重型 IDE。
- **Vim**：极速、命令行之王。
- **Jupyter**：数据科学、交互式。

### 2. 图表绘制 (Diagram)

- **Draw.io / Visio**：标准架构、流程图。
- **Excalidraw**：手绘风、快速草图。
- **Mermaid / PlantUML**：**代码绘图** (Text-to-Chart)。
- **Miro**：在线协作白板。

### 3. AI 工具 (AI)

- **ChatGPT / Claude**：通用辅助、逻辑推导。
- **GitHub Copilot / Tabnine**：代码自动补全。
- **Ollama**：**本地**运行大模型。
- **Midjourney / Stable Diffusion**：AI 绘图。

### 4. 托管与部署 (Cloud)

- **AWS**：全能云基础设施。
- **Cloudflare**：CDN、安全、边缘计算。
- **GitHub**：代码托管 + CI/CD (Actions)。
- **DigitalOcean / Fly.io**：开发者友好的快速部署。

### 5. 代码质量 (Quality)

- **ESLint**：静态扫描、规范检查。
- **Jest**：JS 单元测试。
- **SonarQube**：深度代码质量分析。
- **Selenium**：端到端 (E2E) 自动化测试。

### 6. 安全 (Security)

- **1Password / LastPass**：密码管理。
- **Snyk**：漏洞扫描（代码/依赖）。
- **Nmap**：网络探测与扫描。
- **OWASP**：安全标准与指南。

### 7. 知识管理 (Notes)

- **Notion**：全能文档、协作。
- **Obsidian / Logseq**：**双向链接**、本地 Markdown、第二大脑。
- **Markdown**：文档通用标准。

### 8. 视觉设计 (Design)

- **Figma**：UI/UX 协作首选。
- **Photoshop / Illustrator**：位图/矢量图权威。
- **Canva**：轻量级、模板化设计。
