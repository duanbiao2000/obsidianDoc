# ★ 每日执行力 5 分钟自检清单

## 一、目标清晰度（Goal Clarity）

- [ ]  今天是否有「一个明确目标」？-讲笔记降至600篇以内
- [ ]  目标是否可被具体行动触达？  -日期排序, 删掉/归档过时笔记. 
    ❗ 不清楚目标 → 自动失败日

## 二、行动兑现度（Execution）

- [ ]  今天是否完成 ≥1 个关键行动？(笔记精简, Github star归类优化)
- [ ]  行动是否直接服务于目标？  (笔记建链<-精简)  (源码阅读<-star归类)
    ❗ 忙碌≠推进

## 三、纪律稳定性（Discipline）

- [ ]  是否按计划行动而非凭感觉？(加入了todo reminds, templater)
- [ ]  是否克服了至少一次拖延？  
    ★ 纪律=在不想做时仍去做

## 四、反馈与修正（Feedback）

- [ ]  今天是否获得反馈（结果/他人/数据）？ (直播, 但是言辞不够简练. 三思而后行, 重视对他人的启发与实用性)
- [ ]  是否据此微调策略？  
    ❗ 无反馈=盲目重复

## 五、挫折处理（Resilience）

- [ ]  是否把挫折当作信息而非借口？
- [ ]  是否避免“情绪化放弃”？  
    ★ 失败=信号，不是判决

## 六、信念对齐（Belief Alignment）

- [ ]  今天的信念是否推动我前进？
- [ ]  是否识别并修正自我设限想法？  
    ❗ 错误信念→隐性自毁

## 七、资源调用（Leverage）

- [ ]  是否主动寻求帮助/工具？(perplexity + chatgpt + claude + 通义)
- [ ]  是否避免“硬扛式低效”？  (不追求完美, 多轮迭代)
    ★ 求助=成熟，不是弱点

---

## ★ 快速评分（30 秒）

- 6–7 项达成：高质量执行日 ✅
- 4–5 项达成：可接受 ⚠️
- ≤3 项达成：系统性偏离 ❌

## ★ 一句话复盘

> 今天阻碍我执行的「唯一因素」是？
