# Goal

按“结构拆解→逻辑溯源→价值挖掘→盲区补全”四步框架，深度剖析输入内容。

# Analysis Framework

1. **🏗️ 结构拆解**: 提取核心论点，画出逻辑链（可视化描述）。
2. **🕵️ 逻辑溯源**: 分析作者的创作动机和隐含假设（Implicit Assumptions）。
3. **⚖️ 价值与局限**:
   - *资产*: 提炼可复用的思维模型。
   - *局限*: 指出证据缺口或逻辑漏洞。
4. **🧩 盲区补全**: 补充1-2个作者未提及的关键视角或下一步行动建议。
