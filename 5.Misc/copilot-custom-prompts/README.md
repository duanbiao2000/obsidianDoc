## 🔍 三种检索方式

### 方式一: 前缀过滤 (最快)

在 Obsidian 文件浏览器或搜索框中输入分类代码即可快速过滤:

```
输入 "KM-"  → 显示所有知识管理类提示词
输入 "TH-"  → 显示所有思维框架类提示词
输入 "RL-"  → 显示所有角色扮演类提示词
```

**示例场景**:
- 想优化笔记结构 → 搜索 `KM-` 或 `KM-笔记`
- 需要提问框架 → 搜索 `TH-提问` 或 `TH-苏格拉底`
- 模拟CEO视角 → 搜索 `RL-CEO`

---

### 方式二: MOC 索引 (最详细)

打开对应的 MOC (Map of Content) 文件,按子分类浏览:

| MOC 文件 | 用途 |
|----------|------|
| [[MOC_知识管理类提示词]] | Zettelkasten、费曼笔记、笔记重构 |
| [[MOC_思维框架类提示词]] | 苏格拉底提问、逻辑推理、系统思维 |
| [[MOC_角色扮演类提示词]] | CEO、心理医生、技术专家 |
| [[MOC_技术开发类提示词]] | 代码解释、系统架构、技术文档 |
| [[MOC_写作表达类提示词]] | 学术写作、故事创作、表达优化 |
| [[MOC_学习记忆类提示词]] | Anki闪卡、记忆方法、学习系统 |

MOC 文件包含:
- 子分类组织
- 热门提示词推荐
- 详细功能说明
- 交叉引用链接

---

### 方式三: Dataview 查询 (最灵活)

使用 Dataview 插件进行高级查询,参考 [[Dataview查询模板]]。

**快速示例**:

\`\`\`dataview
# 查询所有知识管理类提示词
LIST
FROM "5.Misc/copilot-custom-prompts"
WHERE contains(file.name, "KM-")
SORT file.name ASC
\`\`\`

\`\`\`dataview
# 按使用频率排序
TABLE file.link as "提示词", view-count as "使用次数"
FROM "5.Misc/copilot-custom-prompts"
WHERE view-count
SORT view-count DESC
LIMIT 10
\`\`\`

## 💡 使用技巧

### 技巧1: 组合搜索
```
"KM" + "卡片"  → 知识管理类卡片相关
"RL" + "CEO"   → CEO角色相关
"TH" + "逻辑"  → 逻辑思维相关
```

### 技巧2: 使用星标收藏
在常用提示词的 YAML 中添加 `star: true`,然后用 Dataview 查询:
\`\`\`dataview
TABLE file.link as "星标提示词"
FROM "5.Misc/copilot-custom-prompts"
WHERE star
\`\`\`

### 技巧3: 记录使用频率
提示词 YAML 中的 `view-count` 字段会自动更新,可用于:
- 发现最受欢迎的提示词
- 清理长期未使用的提示词

### 技巧4: 自定义别名
如果文件名太长,可以在 YAML 中添加 `aliases`:
\`\`\`yaml
---
aliases: [卡片生成, ZK卡片]
---
\`\`\`
然后可以通过别名搜索。

## 🔧 维护指南

### 添加新提示词

1. 确定分类代码 (参考代码体系表)
2. 命名格式: `{代码}-{文件名}.md`
3. 添加 YAML frontmatter
4. 更新对应 MOC 文件

**示例**:
\`\`\`yaml
---
copilot-command-slash-enabled: true
tags:
  - #Domain/CopilotPrompts
  - #Type/Prompt
view-count: 0
update: 2026-01-24
---
\`\`\`

### 更新 MOC 索引

当添加新提示词时,记得在对应 MOC 文件中添加链接:
- 在相应子分类下添加 `- [[新提示词名]] - 描述`
- 更新数量统计
