# 七维度笔记深度梳理提示词模板

## 核心提示词

```
使用七种方法对 {activeNote} 进行深度梳理，目标受众为 {targetAudience}。

七种方法及应用顺序：
1. 分析(Analysis) → 分解结构，识别关键概念、论点、证据
2. 抽象(Abstraction) → 提取核心思想和普遍原则
3. 对比(Comparison) → 比较不同概念/观点的异同
4. 综合(Synthesis) → 连接不同部分，形成新见解
5. 系统化(Systematization) → 组织成逻辑清晰的结构
6. 概括(Summarization) → 简明扼要地总结内容
7. 具体化(Concretization) → 为抽象概念提供具体例子

应用流程：
分析 → 抽象/对比 → 综合/系统化 → 概括/具体化

输出结构（H2-H6分层）：
- 核心概括：主要内容的简要总结
- 关键概念分析：核心概念及其普遍意义
- 结构与关系：逻辑结构和概念联系
- 对比与洞见：对比点和新见解
- 实例与应用：具体例子和应用场景
- 核心洞见：最重要的理解和启示
```

## 应用指南

### 分层输出建议

**初级理解**（快速模板）

- 用时：10-15分钟
- 输出：核心概括 + 关键概念 + 结构 + 3个洞见
- 适合：快速掌握内容要点

**中级理解**（标准模板的前4部分）

- 用时：30-45分钟
- 输出：完整的分析、概念、结构、对比、应用
- 适合：系统学习和笔记梳理

**深度理解**（完整模板）

- 用时：60-90分钟
- 输出：全部7部分，包含方法应用记录
- 适合：知识体系建立和深度研究

### 按笔记类型调整

**概念类笔记**

- 重点：抽象（提取普遍原则）+ 具体化（提供例子）
- 弱化：对比（如无对标概念）

**方法论笔记**

- 重点：系统化（组织步骤）+ 具体化（真实案例）
- 弱化：抽象（已是高度抽象）

**观点论述笔记**

- 重点：分析（论证链条）+ 对比（不同观点）+ 综合（新见解）
- 弱化：系统化（非线性内容）

**实践总结笔记**

- 重点：概括（核心经验）+ 具体化（具体做法）
- 弱化：抽象（已是具体）

### 按学习阶段调整

**入门阶段**

- 使用快速模板
- 重点关注：核心概括、关键概念、基本应用
- 后续可补充对比和洞见

**中阶段**

- 使用标准模板
- 均衡运用七种方法
- 重点建立概念之间的关系网络

**深入阶段**

- 使用完整模板
- 强化综合和洞见部分
- 关注隐含假设和边界条件

## 质量检查清单

梳理完成后验证：

**分析维度**

- [ ] 识别了所有关键概念
- [ ] 列出了主要论点和支撑证据
- [ ] 论证链条清晰

**抽象维度**

- [ ] 提取了可泛化的原则
- [ ] 原则表述清晰准确
- [ ] 说明了普遍意义

**综合维度**

- [ ] 连接了不同的观点或知识
- [ ] 产生了新的见解
- [ ] 新见解有逻辑支撑

**对比维度**

- [ ] 选择了合适的对标
- [ ] 差异点明确
- [ ] 说明了各自的适用场景

**系统化维度**

- [ ] 逻辑结构清晰
- [ ] 概念关系明确
- [ ] 层级递进合理

**概括维度**

- [ ] 核心观点精准
- [ ] 一句话总结准确
- [ ] 要点全面但简洁

**具体化维度**

- [ ] 例子与概念相关
- [ ] 例子足够具体和真实
- [ ] 例子有代表性

**整体维度**

- [ ] 各部分之间有逻辑联系
- [ ] 核心洞见是全面梳理的自然结论
- [ ] 结构清晰，易于理解和查阅

## 常见误区与纠正

|误区|表现|纠正|
|---|---|---|
|**分析过度细节化**|列举所有细节而非关键点|聚焦能支撑核心论点的细节|
|**抽象过度抽离**|原则太宽泛，失去指导意义|在具体和抽象之间找平衡|
|**对比选择不当**|拿不相关的东西对比|选择有可比性的对标对象|
|**综合显得生硬**|强行连接，逻辑不通|基于实际关联而非强行联系|
|**系统化过度复杂**|图表多，反而难以理解|优先用简洁的层级，必要时才用图|
|**概括丢失信息**|总结得太简洁，重点不明|在简洁和完整之间平衡|
|**例子不够具体**|"比如说..."后无实例|给出真实的、详细的案例|
|**缺少核心洞见**|最后部分是各部分的重复|提出之前未出现过的高阶洞见|

## 迭代优化流程

**第一轮**：快速梳理

- 目的：快速掌握笔记概貌
- 输出：核心概括、关键概念、基本结构

**第二轮**：深度分析

- 目的：细化各维度内容
- 重点：补充论证链条、对比分析、新见解

**第三轮**：具体化补充

- 目的：增加可应用性
- 重点：为抽象概念补充真实例子

**第四轮**：核心洞见提炼

- 目的：形成最终理解
- 重点：从各部分提升到更高的认知水平

**第五轮**：质量检查和优化

- 目的：确保结构完整、逻辑清晰
- 重点：消除冗余、加强关联、打磨表述
