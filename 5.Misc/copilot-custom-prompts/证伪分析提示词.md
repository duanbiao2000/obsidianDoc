## 🔍 证伪分析提示词 (Falsification Analysis Prompt)

### **核心指令**

请对 `{activeNote}` 中的**核心观点和论证逻辑**进行系统性的证伪分析。你需要扮演"魔鬼的代言人"(Devil's Advocate),主动寻找可能推翻当前笔记结论的反例、逻辑漏洞和隐含假设。

***

### **分析框架**

#### **第一步: 提取可证伪的核心主张**

从笔记中识别并明确列出:
1. **核心假设** - 笔记隐含但未明说的前提条件
2. **因果关系声明** - 声称A导致B的推理链条
3. **普遍性结论** - 使用"总是"、"必然"、"所有"等绝对化表述的观点

**输出格式:**
```
核心主张1: [用一句话概括]
核心主张2: [用一句话概括]
...
```

***

#### **第二步: 构建证伪测试问题**

针对每个核心主张,生成至少3个证伪性问题,遵循以下模板:[1][2]

**模板A: 反例探寻**
- "在什么情况下,[主张]会失效?"
- "历史上是否存在违背[主张]的真实案例?"
- "如果[反向条件成立],该主张是否依然有效?"

**模板B: 边界条件测试**
- "[主张]适用的范围是什么?在边界外会发生什么?"
- "该主张是否存在隐含的时间/地域/人群限制?"

**模板C: 替代解释挑战**
- "除了[笔记给出的解释],是否存在同样合理的替代理论?"
- "相关性是否等于因果性?是否存在第三变量?"

**示例输出:**
```
证伪测试 - 针对"证伪主义能克服确认偏误":
1. 在哪些情况下,主动寻找反例反而会强化偏见?(如选择性证伪)
2. 心理学研究是否发现证伪策略在普通人中难以执行的证据?
3. 是否存在某些领域,证实比证伪更有效?(如早期探索性研究)
```

***

#### **第三步: 识别隐含假设与逻辑漏洞**

分析笔记时,特别关注:[3][4]

1. **未说明的前提** - 哪些假设被默认为真但未被论证?
2. **逻辑跳跃** - 推理链条中是否存在省略的中间步骤?
3. **样本偏差** - 举例是否具有代表性?是否存在幸存者偏差?
4. **术语模糊** - 关键概念是否有多重解释?是否存在偷换概念?

**输出格式:**
```
隐含假设:
- [假设1]: 笔记默认投资者"有能力"识别反例,但未考虑认知负荷限制
- [假设2]: ...

逻辑漏洞:
- 跳跃点: 从"主动寻找反例"到"决策更稳健"之间,缺少对执行成本的讨论
- ...
```

***

#### **第四步: 提供建设性的证伪建议**

不仅仅指出问题,还需给出如何**强化**或**修正**原笔记观点的路径:

1. **补充限定条件** - 在什么边界内该观点成立?
2. **增加反例讨论** - 承认哪些情况下观点不适用
3. **提出可验证的预测** - 如果观点正确,应该观察到什么现象?
4. **推荐对冲策略** - 如何在实践中降低该观点错误的风险?

**示例输出:**
```
强化建议:
1. 补充条件: "证伪主义在[理性决策者+低情绪环境]中最有效"
2. 承认反例: "在时间紧迫的决策场景下,证实策略可能更高效"
3. 可验证预测: "长期跟踪显示,使用证伪清单的投资者回撤幅度应更小"
4. 对冲策略: "结合贝叶斯更新,允许渐进式调整信念而非二元证伪"
```

***

### **输出要求**

**语言风格:**
- 保持客观中立,避免人身攻击式的批判
- 使用"如果...那么..."句式进行条件性推理
- 提供具体案例而非抽象指责

**结构要求:**
- 遵循"第一步→第四步"的顺序输出
- 每个证伪问题后附上1-2句简要说明
- 最终提供一个综合性的"证伪强度评级"(1-10分)

**长度限制:**
- 总输出控制在800-1200字符
- 每个步骤平衡展开,避免单一部分过长

***

### **评估标准 (自我检查清单)**

完成证伪分析后,请确认:
- [ ] 是否识别出至少3个可证伪的核心主张?
- [ ] 每个主张是否对应至少3个证伪测试问题?
- [ ] 是否指出了至少2个隐含假设?
- [ ] 建议是否具有可操作性(而非仅指出问题)?
- [ ] 分析是否保持了建设性(目标是改进而非推翻)?

***

### 🎯 **立即执行**

现在,请将此证伪分析框架应用于以下笔记内容:

```
{activeNote}
```

开始你的系统性证伪分析。
