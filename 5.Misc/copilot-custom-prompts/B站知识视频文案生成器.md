# B站知识视频文案生成器

---
- **核心逻辑**：将高密度知识（系统 2）转化为低摩擦叙事（系统 1）。
- **本质转变**：从“知识堆砌”转向“认知重构”，通过 Hook-Insight-Action 链条驱动用户留存。
- **成功公式**：反直觉视角 + 深度原理 + 结构化转场 = 高权重视频。

### 2. 提示词内核 (The Meta-Prompt Kernel)

```python
def generate_script(topic, expert_role):
    # 1. 设定语义锚点
    audience = define_audience_profile() # 认知水平、兴趣点
    tone = set_narrative_style(expert_role) # 如：李永乐式逻辑 + 罗翔式幽默
    
    # 2. 结构化生成
    hook = create_high_friction_opening(topic) # 制造认知失调
    body = decompose_logic(topic, sections=3) # 分而治之，层层递进
    
    # 3. 约束与修正
    constraints = [
        "禁止陈词滥调",
        "术语必配通俗比喻",
        "每 2 分钟一个认知反转"
    ]
    
    return compile_script(hook, body, constraints)
```
**认知失调 = 心理矛盾 → 引发不适 → 驱动行动（思考、改变、点击、阅读）**。

### 3. 风格约束与避坑指南

- **能实装不预测**：在文案中直接展示案例/数据，而非泛泛而谈。
- **意图密度**：删除所有“大家以后可以关注一下”类的废话，每一句必须承载信息或情绪。
- **语言特征**：
    - **逻辑性**：因果链条严密，使用“为什么”驱动“怎么做”。
    - **生动性**：每段逻辑后接一个“这就像是...”的类比。
    - **节奏感**：短句控节奏，长句析原理，快慢交替。
