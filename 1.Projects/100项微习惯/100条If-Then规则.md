下面这 100 条 If–Then 原则，都偏向“个人成长 + 情绪管理 + 深度工作”，你可以当素材库，挑适合自己的抄进笔记，用 1–2 周实验一批。

---

## 一、早晨与每日启动（10 条）

1. If 我起床超过 10 分钟还没坐到书桌前, then 立刻坐下打开今日计划。
    
2. If 刚坐到书桌前, then 先花 3 分钟写下今天最重要的 3 件事。
    
3. If 起床后第一时间想摸手机, then 先喝一杯水并做 10 次深呼吸。
    
4. If 今天是工作日的早上 7:00–9:00, then 不刷社交媒体, 只允许查必要信息。
    
5. If 刚打开电脑, then 先打开待办工具, 不先开聊天软件。
    
6. If 早上感觉迷茫, then 写一段 3 行日记：今天我最想推进的 1 件事是什么。
    
7. If 洗漱结束准备出门/开始工作, then 用 30 秒回顾今天的时间安排。
    
8. If 我在早上感到状态差, then 先做 5 分钟拉伸再开始工作。
    
9. If 今天起床时觉得很丧, then 给自己写一条鼓励短信发到备忘录。
    
10. If 今天有重要任务, then 在早上给它预留连续 90 分钟不被打断的时间。

## 二、深度工作与学习（20 条）

11. If 准备开始学习/写作, then 先设 25 分钟倒计时, 期间只做这一件事。
    
12. If 倒计时开始, then 关闭手机通知并把它放到看不到的地方。
    
13. If 学习时想起别的要做的事, then 写在旁边纸上而不是立刻去做。
    
14. If 注意力开始分散, then 先坚持到本轮倒计时结束再休息。
    
15. If 一个任务连续被我拖延 3 天, then 拆成 15 分钟就能完成的最小子任务。
    
16. If 在文档中卡住 10 分钟还写不出内容, then 改成先列出小标题和要点。
    
17. If 我在看学习视频/课程, then 同时打开笔记, 每 5 分钟至少写一条要点。
    
18. If 打开浏览器准备学习, then 先打开教材/论文, 再打开其他网站。
    
19. If 在资料海洋中乱跳 10 分钟, then 立刻停下来写一句：“我现在具体要解决的问题是什么？”
    
20. If 学习结束前 5 分钟, then 写一段“今天我学到了什么/明天接着从哪开始”。
    
21. If 准备开始一个大项目, then 先写一页项目简要：目标、截止日期、三大里程碑。
    
22. If 我开始新一轮深度工作, then 提前决定本轮结束时“可见的产出是什么”。
    
23. If 思考问题超过 15 分钟没进展, then 画一张图或流程图重构思路。
    
24. If 在学习中产生强烈挫败感, then 写下“我具体卡在了哪个概念或步骤”。
    
25. If 我想要拖延难题, then 先承诺只处理 5 分钟, 5 分钟后再决定要不要继续。
    
26. If 我连续 3 次检查邮件/消息间隔小于 15 分钟, then 接下来的 30 分钟强制离线。
    
27. If 今天必须完成的关键任务还没开始, then 不允许开始任何新任务。
    
28. If 明天有重要输出（演讲/报告/考试）, then 今晚至少做 30 分钟针对性准备。
    
29. If 结束一天工作, then 回顾是否有 1 段至少 60 分钟的深度专注时间, 没有就反思原因。
    
30. If 发现自己在原地瞎优化细节, then 问自己：“这一步对最终结果的贡献 > 10% 吗？否则立刻停。”

## 三、手机与分心管理（15 条）

31. If 发现自己无意识解锁手机, then 先问一句“我现在具体想要什么信息？”，答不出就锁回去。
    
32. If 准备滑动短视频, then 先把倒计时设为 10 分钟, 时间到立刻退出。
    
33. If 在工作区, then 手机默认放在够不到的位置。
    
34. If 每小时内打开社交媒体超过 2 次, then 接下来的 2 小时只允许电脑端使用必要通讯。
    
35. If 准备睡前看手机, then 先把第二天早上的第一件重要任务写下来。
    
36. If 在等电梯/排队时想刷手机, then 改为做 10 次深呼吸或观察周围细节。
    
37. If 上厕所时带了手机, then 不打开任何社交 App, 只允许阅读长文或电子书。
    
38. If 刷手机超过 20 分钟, then 接下来必须做 10 分钟整理/收纳/运动。
    
39. If 在工作时间看到非紧急消息, then 标记为稍后处理, 不立刻回复。
    
40. If 午餐时间刷手机, then 每隔 5 分钟问自己一次：“这对我有长期价值吗？”
    
41. If 打开视频网站, then 先搜自己计划看的内容, 不看首页推荐。
    
42. If 上午 9 点前或晚上 10 点后, then 不在手机上处理工作 IM。
    
43. If 点开购物 App, then 先看“购物车里的东西是否还需要”, 不立即搜索新东西。
    
44. If 想在工作中偷偷刷手机, then 先去倒一杯水或走一小圈再回来。
    
45. If 一天手机屏幕时间超过自己设的上限, then 第二天自动降低 30 分钟作为惩罚。

## 四、健康与运动（15 条）

46. If 到了整点, then 站起来活动至少 1 分钟。
    
47. If 午餐吃得很撑, then 午后散步 10–15 分钟。
    
48. If 今天完全没运动, then 睡前做 10 个深蹲 + 10 个俯卧撑/墙壁俯卧撑。
    
49. If 早上起床感觉困倦, then 先拉伸 3 分钟再喝咖啡。
    
50. If 晚上吃完饭还在玩手机, then 先起来走 5 分钟再坐下。
    
51. If 今天坐着工作超过 2 小时, then 强制做一组眼保健操或远眺 2 分钟。
    
52. If 下午感到脑子昏沉, then 优先选择喝水而不是咖啡或甜饮料。
    
53. If 准备点外卖, then 先在备注里加一条“少油少盐”或选择一个更健康的菜。
    
54. If 晚上 11 点还没准备睡觉, then 不再开始任何需要高专注的任务。
    
55. If 上床后还想刷手机, then 打开电子书, 阅读 10 页纸质类内容。
    
56. If 今天压力很大, then 睡前做 5 分钟伸展或冥想。
    
57. If 周末完全待在室内, then 至少安排一次 30 分钟户外散步。
    
58. If 觉得今天吃得很不健康, then 明天主动安排一顿高蔬菜高蛋白的餐。
    
59. If 下午 4 点后想喝含咖啡因饮料, then 先喝一大杯水再决定。
    
60. If 体重/健康数据让自己焦虑, then 写下 1 条本周可以做到的小行动, 不再刷其他“焦虑内容”。

## 五、情绪与人际（15 条）

61. If 感到生气想回消息/发朋友圈, then 先等 10 分钟, 写草稿不直接发送。
    
62. If 和人争执时声音开始变大, then 暂停 10 秒, 用一句“我现在有点激动, 先缓一下”。
    
63. If 一整天没和重要的人说话, then 晚上发一条简单的问候或语音。
    
64. If 收到批评/负面反馈, then 先写下“对方说的 1 点可能有道理的地方”。
    
65. If 感到孤独或压力大, then 主动约一位信任的人线上/线下聊 15 分钟。
    
66. If 想八卦别人, then 改为问一句：“这件事对我有什么启发？”
    
67. If 在社交场合不知说什么, then 提一个开放问题：“最近你在忙什么让你觉得有意思的事？”
    
68. If 和人道别前 1 分钟, then 说一句具体的感谢或鼓励。
    
69. If 被人夸奖, then 简单说“谢谢, 我会继续努力”, 不急着自我否定。
    
70. If 一天中出现强烈情绪, then 晚上用 3 行日记记录：事件–情绪–我学到了什么。
    
71. If 家人/伴侣在说话, then 暂停手上的手机/电脑, 注视对方回应一句后再继续做事。
    
72. If 想在回复里“赢过对方”, then 问自己：“我更想赢这次对话, 还是这段关系？”
    
73. If 意识到自己说话太快, then 刻意在每一句话后停 1 秒。
    
74. If 今天遇到一个让人不舒服的人, then 至少在心里说一句“谢谢你提醒我不要成为这样的人”。
    
75. If 和朋友聊完感到能量被抽空, then 以后减少临时答应对方请求, 更早设边界。

## 六、金钱与决策（10 条）

76. If 想买非必需品, then 放进购物车至少 24 小时再决定。
    
77. If 想冲动消费, then 写下“我想通过这件东西获得什么感觉？有没有更便宜的方式获得？”
    
78. If 本月开销接近预算上限, then 暂停所有娱乐/冲动型消费 3 天。
    
79. If 看到某个“暴富机会”, then 至少找 2 个独立信息源验证后再考虑。
    
80. If 想加仓/投资, then 写下“如果它继续跌 30% 我是否还睡得着”。
    
81. If 一个订阅服务连续 2 个月几乎不用, then 立刻取消订阅。
    
82. If 今天收入有意外增加, then 至少拿出 20% 存入备用金/投资账户。
    
83. If 有人向我推销金融产品, then 要求对方用 3 句话解释清楚收益来源和主要风险。
    
84. If 为钱做决定感到很不安, then 不当场答应, 至少给自己 24 小时思考。
    
85. If 想借钱给朋友/家人, then 只借自己心理上可以“当作送掉”的数额。

## 七、自我反思与长期成长（15 条）

86. If 一天结束前, then 写下今天最值得感激的 1 件事和最想改进的 1 件事。
    
87. If 连续一周执行不了某条 If–Then 规则, then 修改它, 让动作更小更具体。
    
88. If 一件事让我后悔, then 分析是“信息不足”“情绪失控”还是“价值观模糊”。
    
89. If 我发现自己在抱怨, then 加一句：“那我能做的一小步是什么？”
    
90. If 刷到让我焦虑的成功案例, then 写下对方背后可能付出的看不见的成本。
    
91. If 一周内完全没有“不舒服的成长体验”, then 主动安排一次有挑战性的任务。
    
92. If 遇到让自己羡慕的人, then 问自己“我可以向 TA 学 1 件具体习惯是什么？”
    
93. If 一件事我拖延超过一个月, then 要么安排具体日期执行, 要么明确写下“我选择不做”并接受。
    
94. If 做完一件很重要的事情, then 留出 10 分钟写下过程中的有效策略和坑。
    
95. If 感到“人生很乱”, then 抽 30 分钟画出自己的 3–5 个长期方向和当前的主要项目。
    
96. If 自己犯错了, then 尝试在 24 小时内主动向受影响的人解释并道歉。
    
97. If 发现自己在逃避一个事实, then 至少在日记里用一句话如实写出来。
    
98. If 一周结束, then 复盘：这周哪一条 If–Then 对我帮助最大, 哪一条需要删掉或调整。
    
99. If 未来有重要选择, then 预先写出“若选 A, 我愿意承担的 3 个代价”。
    
100. If 感到对未来恐惧, then 把恐惧写成具体情景, 并为每个情景写一个最小应对动作。
