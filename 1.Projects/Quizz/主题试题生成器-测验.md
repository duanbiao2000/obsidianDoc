### ✅ 基础架构（1-10）

**1. LangChain 的核心设计理念是：**

**2. LangChain 中的 `Chain` 是指：**

C. 一种将多个模块串联的逻辑结构

B. HuggingfacePipeline  
C. OpenAI LLM  
D. RasaModel

**4. LangChain 是用什么语言实现的？**  
A. Java  
B. Go  
C. Python  
D. Rust

**5. LangChain 的 Prompt 模块作用是？**  
A. 自动生成提示词  
B. 控制模型并发  
C. 构建和格式化输入模板  
D. 编写 API 接口

**6. LangChain 的文档加载组件属于哪个模块？**  
A. LLMs  
B. Tools  
C. Document Loaders

**8. LangChain 支持的主流 LLM 接入方式不包括：**  
A. OpenAI  
B. Anthropic  
C. Huggingface Hub

**9. LangChain 的 `LLMChain` 中，核心执行组件是？**
