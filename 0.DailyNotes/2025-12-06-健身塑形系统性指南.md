**第1-2周：基础适应**

- 目标：建立运动习惯，提高心肺耐受
- 周频率：4次训练（例：Mon胸，Wed腿，Fri背，Sat有氧+核心）
- 强度：60-65% MHR，中等负重，12-15次
- 有氧：每次20-30分钟，中低强度
- 营养：热量赤字300kcal，蛋白质1.8g/kg

**第3周：强度提升**

- 目标：增加热量消耗，开始看到变化
- 周频率：5次训练（加入额外有氧日）
- 强度：65-75% MHR，中等负重，10-12次，短间隔（45秒）
- 有氧：加入1次高强度间歇训练（HIIT，20分钟）
- 营养：热量赤字400kcal

**第4周：恢复与评估**

- 目标：身体恢复，为下一周期做准备
- 周频率：3-4次训练（轻强度）
- 强度：60% MHR，轻负重，恢复性训练
- 有氧：低强度，20分钟
- 营养：恢复至维持热量，高蛋白

#### 增肌周期（4周）

**第1-2周：基础卷**

- 目标：建立力量基础
- 周频率：4次训练（上/下分化）
  - 周一、四：上身（胸、背、肩）
  - 周三、六：下身（腿）
- 强度：8-10 RM，充分休息（2-3分钟）
- 有氧：最少（10-15分钟热身）
- 营养：热量盈余300kcal，蛋白质1.8g/kg

**第3周：高容量卷**

- 目标：提高肌肉泵感和代谢应激
- 周频率：4次训练
- 强度：10-12 RM，中等休息（90秒）
- 增加1-2个孤立动作
- 有氧：最少
- 营养：热量盈余400kcal

**第4周：减量卷**

- 目标：恢复神经系统，防止过度训练
- 周频率：3次训练（轻强度）
- 强度：降低50%体积和强度
- 有氧：可做30分钟低强度
- 营养：恢复至维持热量

### 六、动作纠正视觉指南（文字版）

#### 平板支撑的正确形式检查

**正确姿态**

- 头：与躯干成直线，眼睛向下看
- 肩膀：在肘部正上方，肩胛骨下沉（不耸起）
- 躯干：完全平直，不下沉，不上翘
- 核心：收紧，想象腹肌向脊柱靠近
- 臀部：与躯干齐平（不下沉，不过高）
- 腿：绷直，脚尖着地

**常见错误及修正**

1. **臀部下沉** → 放在镜子前，核心意念集中，短时间多组（3组30秒比1组90秒更有效）
2. **肩膀耸起** → 做肩胛骨下沉激活，每组前做5次
3. **腰部突起** → 减少时间，加强核心激活

---

#### 深蹲的正确形式检查

**准备**

- 脚肩宽分开，脚尖微外(5-10度)
- 胸部挺起，肩膀后缩
- 核心收紧，脊柱中立

**下蹲过程**

- 髋部和膝盖同时弯曲
- 膝盖与脚尖方向一致（向前或微外）
- 胸部始终挺起，不前倾过度
- 脚跟贴地（整个过程）

**底部位置**

- 髋部低于膝盖（或同高）
- 脊柱中立（不过度弯曲）

**上升过程**

- 用腿部推起，不用腰部
- 膝盖和髋部同时伸展

**常见错误及修正**

1. **膝盖内扣** → 弹力带横向走激活臀肌，做蛤壳式拉伸
2. **脚跟离地** → 抬脚跟垫子或做踝关节灵活性训练
3. **躯干过度前倾** → 做背部灵活性训练，背靠墙做深蹲

## 总结与使用建议

### 对教练的建议

1. **个性化评估**：在第一堂课时，用自检表评估学员的呼吸、流畅性和核心稳定性，找出主要瓶颈。
2. **循序渐进**：不要急于增加重量或难度，优先确保形式完美。
3. **反馈频率**：每组后给予1-2个具体改进点，不要过度纠正。
4. **记录与追踪**：使用日志模板追踪进度，每4周评估一次。
5. **营养指导**：不仅教练习，也要教正确的饮食，这决定了50%的结果。

### 对学员的建议

1. **质量优于数量**：一个完美的深蹲比10个糟糕的深蹲更有价值。
2. **持之以恒**：4周看到身体变化，8周周围人看到，12周改变生活。
3. **记录每次训练**：用日志模板追踪，找到适合自己的强度和节奏。
4. **定期评估**：每4周拍照、量体重、测体脂，确认是否在正轨上。
5. **充足恢复**：训练只是刺激，恢复（睡眠+营养）才是生长的关键。

## 第一部分：核心原理篇

### 一、呼吸控制技术（Breathing Control）

**核心原则**：呼吸不仅是供氧，更是驱动肌肉启动、稳定核心、提高动作质量的关键。

#### 1.1 基础呼吸模式

**胸式呼吸 vs 腹式呼吸**

- **腹式呼吸**（推荐用于训练）：吸气时腹部向外扩张，呼气时腹部内收。优势：能更好地启动核心肌群，稳定脊柱。
- **胸式呼吸**：主要由肋间肌驱动，易导致肩膀紧张。仅在有氧训练高强度阶段适用。

**标准呼吸节奏**：4秒吸气，4秒呼气（基础），可根据动作难度调整为2:2或6:6。

#### 1.2 训练中的呼吸节奏规则

| 训练阶段        | 呼吸方法      | 作用               |
| ----------- | --------- | ---------------- |
| 准备阶段        | 深呼吸3次     | 激活核心，进入专注状态      |
| 离心阶段（放松、下降） | 吸气        | 维持脊柱稳定，防止腰椎过度伸展  |
| 向心阶段（用力、上升） | 呼气        | 启动肌肉，增加腹压，提高力量输出 |
| 静态保持        | 缓慢呼吸，避免屏气 | 防止血压升高，保证氧供      |
| 恢复间隔        | 深腹式呼吸     | 降低心率，恢复核心张力      |

**常见错误**：屏气、呼吸浅促、呼吸与动作不同步 → 导致核心失稳、腰椎代偿、力量下降。

#### 1.3 不同训练目标的呼吸变化

- **减脂训练**：快速呼吸（2:2节奏），配合高心率，增加热量消耗。
- **增肌训练**：缓慢有力的呼吸（4:4或6:6），强调腹压和力量。
- **塑形训练**：中等节奏（3:3），平衡心率和肌肉张力。
- **运动表现**：根据爆发力需求，可短暂屏气（1秒以内），但立即恢复呼吸。

### 二、动作流畅性原则（Movement Flow）

**核心理念**：每个动作不是孤立的，而是一个连贯的身体对话。流畅性反映了神经肌肉协调能力。

#### 2.1 流畅性的三个维度

**1. 时间连贯性（Temporal Continuity）**

- 动作之间无卡顿、无突然加速或减速
- 节奏稳定，就像跳舞一样自然
- 表现：能否在没有外界提示的情况下，自主完成连贯的动作序列

**2. 空间精准性（Spatial Precision）**

- 每个关节在预定的轨迹上移动
- 避免代偿模式（如：做肩推时脊柱过度后伸）
- 表现：核心始终稳定，骨盆/肩胛带不晃动

**3. 能量效率（Energy Efficiency）**

- 最少的肌肉激活完成动作
- 避免多余的动作幅度或震颤
- 表现：动作看起来轻松、有力、不费力

#### 2.2 流畅性的五级进阶

| 级别            | 特征                 | 教练指导               |
| ------------- | ------------------ | ------------------ |
| Level 1：卡顿    | 动作分段、停顿、需要重新启动     | "放慢速度，让身体自然连接各个动作" |
| Level 2：缓慢连贯  | 动作连接但速度很慢，需要高度集中   | "保持这个节奏，逐渐提速"      |
| Level 3：中等流畅  | 正常速度下动作连贯，但偶有卡顿    | "很好，再稳定一次，感受肌肉的节奏" |
| Level 4：高流畅   | 无卡顿，节奏稳定，核心始终稳定    | "这就是完美的形式，现在加大阻力"  |
| Level 5：艺术化流畅 | 像舞蹈一样的优美，力量和柔和完美平衡 | "你可以开始教别人了"        |

#### 2.3 如何改善流畅性

1. **放慢速度**：将每个动作的周期延长至6-8秒，强制神经肌肉系统建立新的协调模式。
2. **感受链接**：在每个关键点（如下降到底部时），停顿1秒，感受肌肉的激活。
3. **消除代偿**：用镜子或视频反馈，识别哪个环节破坏了流畅性。
4. **配合呼吸**：呼吸的节奏往往决定了动作的流畅性，两者必须同步。

### 三、三大训练目标的原理

#### 3.1 减脂训练（Fat Loss）

**生理原理**：通过提高心率、增加热量消耗、优化激素环境来减少体脂。

**训练特征**：

- 高心率（60-80% 最大心率）
- 较少休息（30-60秒）
- 中等阻力 + 快速节奏
- 结合有氧和无氧元素

**激活肌群的优先级**：

1. 大肌群优先（腿、背、胸）→ 单次热量消耗最大
2. 复合动作优先（深蹲、硬拉、推举）→ 多关节参与
3. 核心稳定贯穿全程 → 提高代谢效率

**营养侧重**：热量赤字、高蛋白（维持肌肉）、低精制碳水

#### 3.2 增肌训练（Muscle Building）

**生理原理**：通过机械张力（mechanical tension）、肌肉损伤（muscle damage）、代谢应激（metabolic stress）来刺激肌肉生长。

**训练特征**：

- 中等到高阻力（6-12 RM范围）
- 充分休息（90-120秒）
- 缓慢有控制的动作（3-4秒离心，1-2秒向心）
- 强调"泵感"和肌肉充血

**激活肌群的优先级**：

1. 复合动作（建立肌肉基础）→ 3-4组，6-8 RM
2. 孤立动作（精雕细刻）→ 3组，8-12 RM
3. 高次数小阻力完成（代谢应激）→ 2组，15-20次

**营养侧重**：热量盈余、高蛋白（每公斤体重1.6-2.2g）、充足碳水补充能量

#### 3.3 塑形训练（Body Toning）

**生理原理**：通过增加肌肉耐力、改善肌肉定义、优化体态来改变身体外观。

**训练特征**：

- 中等阻力（8-15 RM）
- 中等休息（60-90秒）
- 匀速动作（2-3秒离心，2-3秒向心）
- 强调肌肉控制和代谢应激的平衡

**激活肌群的优先级**：

1. 目标部位的复合动作 → 3组，8-12次
2. 目标部位的孤立动作 → 3组，12-15次
3. 功能性动作 → 1-2组，全身性

**营养侧重**：维持热量或轻微赤字、中等蛋白、均衡碳脂肪

### 减脂训练

#### 减脂 - 胸部

**训练原则** 胸部是身体最大的肌群之一，高效刺激胸肌能显著提升整体代谢。减脂阶段强调快速、连贯的复合动作，以及短间隔恢复。

**核心动作**

1. **俯卧撑（Push-up）** - 自重或递阶

   - 执行：身体呈直线，胸部接近地面，肘部不锁死
   - 节奏：1秒下降，1秒上升（快速）
   - 组数：3组，12-15次，间隔45秒
2. **哑铃卧推（Dumbbell Bench Press）** - 中等重量

   - 执行：肩胛骨贴紧凳子，腕部中立，肘部约45度外张
   - 节奏：2秒下降，1秒上升
   - 组数：3组，10-12次，间隔60秒
3. **跪姿胸部飞鸟（Kneeling Chest Fly）** - 轻重量

   - 执行：躯干挺直，手臂微弯，胸部主导动作
   - 节奏：2秒打开，1秒合拢
   - 组数：2组，15次，间隔45秒

**呼吸节奏**

- 吸气下降/打开
- 呼气上升/合拢
- 快速节奏（2:2），心率保持在65-75% MHR

**常见错误 & 纠正**

| 错误     | 原因      | 纠正方案                                     |
| ------ | ------- | ---------------------------------------- |
| 肩膀耸起   | 斜方肌代偿   | 在动作前做肩胛骨下沉练习（Scapular Depression），每组前做5次 |
| 腰部过度后伸 | 核心不稳定   | 加强核心激活，每次训练前做30秒平板支撑                     |
| 肘部锁死   | 关节压力过大  | 强调肘部保持微弯，不要追求完全伸展                        |
| 动作幅度过大 | 追求深度而失控 | 减少重量，控制在肘部与肩同高的位置                        |

**训练周期**

- 频率：每周2次（与背部分开）
- 总容量：6-9组有效组
- 递进：每2周增加1-2次或增加1个动作

#### 减脂 - 背部

**训练原则** 背部肌群面积大、代谢高。减脂阶段需要强调纵向和水平拉力的结合，以及快速节奏来维持心率。

**核心动作**

1. **引体向上/辅助引体（Pull-up / Assisted Pull-up）** - 自重或机械辅助

   - 执行：肩胛骨下沉，胸部接近横杆，肘部贴向躯干
   - 节奏：2秒上升，2秒下降
   - 组数：3-4组，力竭（或6-10次），间隔60秒
2. **哑铃划船（Dumbbell Row）** - 中等重量

   - 执行：单膝跪地/站姿，胸部前倾，肘部贴向躯干，肩胛骨后缩
   - 节奏：2秒上升，2秒下降
   - 组数：3组，每侧10-12次，间隔45秒
3. **面拉（Face Pull）** - 轻重量

   - 执行：绳索或弹力带，拉向面部，肘部高于肩，强调后肩启动
   - 节奏：1秒向后，2秒回到起点
   - 组数：2组，15次，间隔45秒

**呼吸节奏**

- 吸气下降/伸展
- 呼气上升/收缩
- 中等到快速（3:2）

**常见错误 & 纠正**

| 错误        | 原因      | 纠正方案                    |
| --------- | ------- | ----------------------- |
| 用手臂而非背部发力 | 肩胛骨没有启动 | 每组前做5次肩胛骨后缩练习，建立神经肌肉连接  |
| 脊柱过度弯曲    | 躯干不稳定   | 核心激活更强，收紧腹部，想象脊柱被拉长     |
| 肩膀不稳定     | 肩部活动度差  | 先做肩部热身动作（臂圈、缓慢划船），增加活动度 |
| 动作范围小     | 用重量太大   | 减轻重量30%，强调充分的肩胛骨运动      |

**训练周期**

- 频率：每周2次（与胸部分开，1-2天间隔）
- 总容量：8-12组有效组
- 递进：增加拉力或增加次数

#### 减脂 - 腿部

**训练原则** 腿部肌群最大，也是代谢最高的区域。减脂阶段需要高心率、高容量、快速恢复。强调下肢爆发力和心血管刺激的结合。

**核心动作**

1. **双腿深蹲（Bodyweight Squat）** - 自重或负重

   - 执行：足肩宽分开，脚尖指向前或微外，膝盖与脚尖对齐，臀部向下、向后，胸部挺起，脊柱中立
   - 节奏：2秒下降，1秒上升（快速）
   - 组数：3组，15-20次，间隔45秒
2. **保加利亚分腿蹲（Bulgarian Split Squat）** - 自重或哑铃

   - 执行：后腿放在凳子上，前腿深蹲，重心在前腿
   - 节奏：2秒下降，1秒上升
   - 组数：3组，每侧10-12次，间隔45秒
3. **直腿硬拉（Stiff-Leg Deadlift）** - 轻哑铃

   - 执行：微屈膝，脊柱中立，髋部铰链向前，感受腿后肌的拉伸
   - 节奏：2秒下降，1秒上升
   - 组数：2组，12-15次，间隔45秒
4. **跳跃深蹲（Jump Squat）** - 自重，爆发力

   - 执行：快速下蹲，爆发式跳起，着地时缓冲，立即进入下一个深蹲
   - 节奏：1秒下蹲，0.5秒跳起
   - 组数：2组，8-10次，间隔60秒

**呼吸节奏**

- 吸气下蹲/下降
- 呼气上升/跳起
- 快速节奏（2:1），保持心率在70-80% MHR

**常见错误 & 纠正**

| 错误     | 原因             | 纠正方案                       |
| ------ | -------------- | -------------------------- |
| 膝盖内扣   | 臀肌激活不足         | 在训练前激活臀肌：横向弹力带走、单腿硬拉、臀桥各1组 |
| 脚跟离地   | 踝关节活动度差或重心过于靠前 | 做踝关节活动度训练，确保脚跟始终贴地         |
| 躯干过度前倾 | 核心弱或活动度问题      | 做竖脊肌和核心激活练习，减轻负重           |
| 背部弯曲   | 脊柱稳定性差         | 先做背部激活和平板支撑，确保脊柱中立         |

**训练周期**

- 频率：每周2-3次（可与上身结合）
- 总容量：10-15组有效组
- 递进：增加负重或增加跳跃次数

#### 减脂 - 肩部

**训练原则** 肩部小肌群，但代谢活跃。减脂阶段强调多向度动作（前、侧、后），快速节奏。

**核心动作**

1. **哑铃肩推（Dumbbell Shoulder Press）** - 中等重量

   - 执行：坐姿或站姿，核心收紧，哑铃置于肩高，向上推起，肘部不锁死
   - 节奏：2秒上升，2秒下降
   - 组数：3组，10-12次，间隔60秒
2. **侧平举（Lateral Raise）** - 轻重量

   - 执行：身体挺直，手臂微弯，侧平举至肩高，肘部始终微弯
   - 节奏：2秒上升，2秒下降
   - 组数：2组，15次，间隔45秒
3. **反向飞鸟（Reverse Pec Deck / Rear Delt Fly）** - 轻到中等

   - 执行：俯身，手臂微弯，向两侧打开，强调后肩启动
   - 节奏：1秒打开，2秒回到起点
   - 组数：2组，15次，间隔45秒

**呼吸节奏**

- 吸气下降/回到起点
- 呼气上升/打开
- 中等快速（2:2）

**常见错误 & 纠正**

| 错误           | 原因          | 纠正方案                    |
| ------------ | ----------- | ----------------------- |
| 腰部参与（脊柱过度后伸） | 核心没有激活或负重过大 | 坐姿做肩推，背部贴椅子，强制脊柱中立      |
| 肩膀耸起         | 斜方肌代偿       | 动作前做肩胛骨下沉练习，每次下降时主动下沉肩膀 |
| 肘部过度外张       | 动作不当        | 强调肘部位置在身体前方，不要完全侧平      |
| 摇晃           | 负重过大        | 减轻重量20%，强调稳定性和控制        |

**训练周期**

- 频率：每周1-2次
- 总容量：6-8组有效组
- 递进：增加重量或增加次数

#### 减脂 - 核心

**训练原则** 核心不仅是美学追求，更是运动表现和脊柱健康的基础。减脂阶段强调动态核心稳定和高代谢动作。

**核心动作**

1. **平板支撑（Plank）** - 自重或递阶

   - 执行：身体呈直线，肩膀在肘部正上方，核心收紧，臀部不下沉
   - 节奏：静态保持，匀速呼吸
   - 组数：3组，30-60秒，间隔45秒
2. **死虫（Dead Bug）** - 自重，动态

   - 执行：仰卧，膝盖弯曲90度，对侧上肢和下肢交替伸直
   - 节奏：2秒伸展，2秒回到起点
   - 组数：3组，10-12次每侧，间隔45秒
3. **山式冲刺（Mountain Climbers）** - 自重，爆发力

   - 执行：从平板支撑姿势开始，快速交替抬膝，保持髋部稳定
   - 节奏：1秒每次膝盖抬起
   - 组数：3组，20次（共40条腿），间隔45秒
4. **木刀式（Pallof Press）** - 轻阻力

   - 执行：侧向站立，抗旋转地向前推，核心抵抗扭转力
   - 节奏：2秒推出，2秒回到胸前
   - 组数：2组，每侧12次，间隔45秒

**呼吸节奏**

- 平板支撑：缓慢腹式呼吸（避免屏气）
- 动态动作：吸气准备，呼气推动
- 快速节奏保持心率

**常见错误 & 纠正**

| 错误        | 原因         | 纠正方案                 |
| --------- | ---------- | -------------------- |
| 平板支撑时臀部下沉 | 核心无力       | 减短时间至20秒，强调质量，逐周增加时间 |
| 平板支撑时臀部过高 | 错误的脊柱位置    | 使用镜子，确保头-躯干-腿呈直线     |
| 山式冲刺时臀部摇晃 | 节奏过快或核心无力  | 降低速度，强调每一次抬膝的质量      |
| 呼吸屏气      | 紧张或不知道如何呼吸 | 教练示范，强调缓慢、持续的腹式呼吸    |

**训练周期**

- 频率：每周3次（可与其他训练结合）
- 总容量：8-10组有效组
- 递进：增加时间或难度（如单腿平板）

### 增肌训练

#### 增肌 - 胸部

**训练原则** 增肌阶段强调机械张力（中等到高负重）和充分的体积（3-4组，8-12次）。胸部需要多向度刺激（按压、飞鸟、倾斜）。

**核心动作**

1. **杠铃卧推（Barbell Bench Press）** - 高负重

   - 执行：肩胛骨贴紧凳子，脚踩地面产生稳定性，杠铃轨迹从胸口到肩正上方，肘部约45度
   - 节奏：3秒下降，1秒暂停，1秒上升（强调离心）
   - 组数：4组，6-8次，间隔2-3分钟
2. **哑铃卧推（Dumbbell Bench Press）** - 中等到高负重

   - 执行：哑铃轨迹比杠铃更深，充分拉伸胸肌，顶部时哑铃不完全接触
   - 节奏：3秒下降，1秒上升
   - 组数：3组，8-10次，间隔2分钟
3. **杠铃上斜卧推（Incline Barbell Bench Press）** - 中等负重

   - 执行：凳子角度30-45度，强调上胸部，肘部角度比平卧推更垂直
   - 节奏：3秒下降，1秒上升
   - 组数：3组，8-10次，间隔2分钟
4. **哑铃胸部飞鸟（Dumbbell Chest Fly）** - 中等负重

   - 执行：肘部微弯，手臂打开呈弧形，强调拉伸和收缩
   - 节奏：3秒打开，1秒暂停，2秒合拢（强调泵感）
   - 组数：3组，10-12次，间隔90秒

**呼吸节奏**

- 吸气下降
- 呼气上升
- 缓慢节奏（4:2或3:1），强调肌肉张力

**常见错误 & 纠正**

| 错误            | 原因           | 纠正方案                              |
| ------------- | ------------ | --------------------------------- |
| 负重过大导致形式不良    | 自尊心或误解"越重越好" | 教练演示，强调"重量服从形式"，宁可减少20%重量也要保证动作质量 |
| 杠铃轨迹偏离（向颈部倾斜） | 肩膀不稳或手腕不中立   | 做肩胛骨和手腕激活，用镜子提供反馈                 |
| 快速动作（爆发式）     | 想追求更大的重量     | 强调增肌阶段就是要"慢"，用计时器或教练数拍子           |
| 肘部锁死（顶部完全伸展）  | 追求充分的伸展      | 保持肘部微弯，避免关节压力                     |
| 不做暂停          | 缩短总时间，减轻痛苦   | 强调暂停的重要性：增加肌肉时间张力，刺激肌肉生长          |

**训练周期**

- 频率：每周1-2次（优先在精力最好的时候）
- 总容量：12-16组有效组
- 递进：每2-3周增加1-2个重量，保持6-8次范围

#### 增肌 - 背部

**训练原则** 背部增肌需要结合纵向拉力（引体向上、下拉）和水平拉力（划船）。重点是充分激活背阔肌和中背部。

**核心动作**

1. **引体向上（Pull-up）** - 自重或负重

   - 执行：肩胛骨下沉，胸部接近横杆，肘部贴向躯干，顶部短暂停顿
   - 节奏：2秒上升，1秒暂停，3秒下降
   - 组数：4组，6-10次，间隔2-3分钟
2. **高位下拉（Lat Pulldown）** - 中等到高负重

   - 执行：胸部微前倾，肘部垂直向下拉，肩胛骨内收，强调背阔肌启动
   - 节奏：2秒下拉，1秒暂停，3秒回到起点
   - 组数：3组，8-12次，间隔90秒
3. **杠铃硬拉（Barbell Deadlift）** - 高负重

   - 执行：脚肩宽，脊柱中立，髋部铰链，杠铃贴身体，用腿和背部同时发力
   - 节奏：3秒上升，1秒暂停，2秒下降
   - 组数：3组，5-8次，间隔3分钟
4. **杠铃划船（Barbell Row）** - 高负重

   - 执行：躯干前倾约45度，杠铃贴身体，肘部贴向躯干，胸部接近杠铃
   - 节奏：1秒上升，1秒暂停，2秒下降
   - 组数：4组，6-8次，间隔2-3分钟

**呼吸节奏**

- 吸气下降/伸展
- 呼气上升/收缩
- 缓慢节奏（4:2或3:1）

**常见错误 & 纠正**

| 错误            | 原因        | 纠正方案                    |
| ------------- | --------- | ----------------------- |
| 用手臂而非背部发力     | 肩胛骨启动不足   | 每组前做5次肩胛骨后缩练习，下拉时先后缩肩胛骨 |
| 躯干过度摇晃        | 负重过大或核心不稳 | 减轻重量，加强核心激活             |
| 脊柱过度弯曲（硬拉/划船） | 活动度差或核心弱  | 做背部灵活性训练，降低重量，强调脊柱中立    |
| 肩膀耸起          | 斜方肌代偿     | 强调肩胛骨下沉，每次下降时主动下沉肩膀     |

**训练周期**

- 频率：每周1-2次
- 总容量：14-18组有效组
- 递进：每2-3周增加负重，保持指定的RM范围

#### 增肌 - 腿部

**训练原则** 腿部增肌需要高负重和高容量的结合。强调深蹲（前、后、哥布林蹲）和硬拉变体，以及腿部孤立动作。

**核心动作**

1. **后蹲（Back Squat）** - 高负重

   - 执行：杠铃放在后肩，脚肩宽，下蹲至膝盖与髋部同高，脊柱中立
   - 节奏：3秒下蹲，1秒暂停，1秒上升
   - 组数：4组，6-8次，间隔2-3分钟
2. **腿举（Leg Press）** - 中等到高负重

   - 执行：脚肩宽，下降至膝盖约90度，臀部贴椅子，不做膝盖超伸
   - 节奏：3秒下降，2秒上升
   - 组数：3组，8-12次，间隔2分钟
3. **罗马尼亚硬拉（Romanian Deadlift）** - 中等负重

   - 执行：微屈膝，髋部铰链向前，感受腿后肌的强烈拉伸，杠铃贴腿
   - 节奏：3秒下降，1秒暂停，1秒上升
   - 组数：3组，8-10次，间隔90秒
4. **腿部伸展（Leg Extension）** - 中等负重，孤立动作

   - 执行：坐姿，腿部向前伸直，顶部1秒停顿，强调股四头肌收缩
   - 节奏：2秒上升，1秒暂停，3秒下降
   - 组数：3组，10-15次，间隔60秒

**呼吸节奏**

- 吸气下降/伸展
- 呼气上升/收缩
- 缓慢节奏（4:2）

**常见错误 & 纠正**

| 错误        | 原因           | 纠正方案                                |
| --------- | ------------ | ----------------------------------- |
| 膝盖内扣（后蹲）  | 臀肌激活不足       | 做弹力带横向走、单腿硬拉激活臀肌                    |
| 脚跟离地      | 踝关节活动度差或重心靠前 | 抬脚跟放垫子（Olympic lifting鞋），或做踝关节活动度训练 |
| 膝盖超伸（腿举）  | 完全锁定膝关节      | 强调保持膝盖微弯，不要完全伸展                     |
| 腰部离椅子（腿举） | 负重过大或核心弱     | 减轻重量，确保臀部始终贴椅子                      |
| 躯干前倾（后蹲）  | 脊柱不稳定或活动度差   | 做背部和髋部灵活性训练，减轻负重                    |

**训练周期**

- 频率：每周1次深蹲 + 1次辅助动作
- 总容量：14-18组有效组
- 递进：每2周增加负重，保持RM范围

1. **杠铃肩推（Barbell Shoulder Press）** - 高负重

   - 节奏：2秒上升，2秒下降
   - 组数：3组，8-10次，间隔2分钟
3. **侧平举（Lateral Raise）** - 轻到中等负重

   - 执行：肘部始终微弯，侧平举至肩高，顶部短暂停顿
   - 节奏：2秒上升，1秒暂停，2秒下降
   - 组数：3组，12-15次，间隔60秒
4. **反向飞鸟（Reverse Pec Deck）** - 中等负重

   - 执行：后肩启动，手臂打开呈弧形，强调后三角肌收缩
   - 节奏：1秒打开，1秒暂停，2秒回到起点
   - 组数：3组，12-15次，间隔60秒

**呼吸节奏**

- 吸气下降
- 呼气上升

| 肩膀耸起         | 斜方肌代偿      | 下降时主动下沉肩膀，做肩胛骨下沉激活 |
| 肘部过度内收       | 肩部活动度差     | 做肩部热身动作增加活动度       |

**训练周期**

- 总容量：12-16组有效组

#### 增肌 - 核心

**训练原则** 增肌阶段的核心训练强调抗阻力和负重，而非仅仅的等距收缩。

**核心动作**

1. **加权平板支撑（Weighted Plank）** - 轻负重

   - 执行：背上放置重板或哑铃，保持脊柱中立，核心最大收缩
   - 节奏：静态保持，每20秒调整一次姿态
   - 组数：3组，40-60秒，间隔60秒
2. **负重腹部卷曲（Weighted Cable Crunch）** - 中等负重

   - 执行：膝盖弯曲，向前弯曲脊柱，感受腹直肌最大收缩
   - 节奏：2秒下降，2秒上升
   - 组数：3组，12-15次，间隔60秒
3. **悬吊腿提（Hanging Leg Raise）** - 自重或负重

   - 执行：双手握杆，腿部向上抬至胸前高度，下降时不摆晃
   - 节奏：2秒上升，2秒下降
   - 组数：3组，8-12次，间隔90秒
4. **旋转木刀式（Weighted Pallof Press）** - 轻到中等负重

   - 执行：抵抗旋转扭力，向前推，保持躯干稳定
   - 节奏：2秒推出，2秒回到胸前
   - 组数：2组，每侧12次，间隔60秒

**呼吸节奏**

- 吸气准备
- 呼气收缩
- 中速节奏（3:2）

**常见错误 & 纠正**

| 错误             | 原因     | 纠正方案         |
| -------------- | ------ | ------------ |
| 负重过大导致脊柱弯曲（卷曲） | 自尊心    | 强调孤立腹直肌，减轻负重 |
| 摇晃（悬吊腿提）       | 下降速度过快 | 放慢速度，强调控制    |
| 屏气             | 紧张     | 教练提醒，强调持续呼吸  |
| 腰部代偿           | 核心启动不足 | 做腹肌激活练习，减轻负重 |

**训练周期**

- 频率：每周2-3次
- 总容量：8-12组有效组
- 递进：增加负重或时间

### 三、常见错误快速纠正指南

#### 核心激活不足

**症状**：腰部过度后伸、骨盆晃动、脊柱不稳定\
**快速纠正**：

1. 在训练前30秒平板支撑 + 20次死虫
2. 每个动作前喊"收紧核心"，让学员主动激活
3. 使用镜子或视频提供即时反馈

#### 肩胛骨代偿（背部训练）

**症状**：用肩膀力量而不是背肌\
**快速纠正**：

1. 做5次肩胛骨后缩激活
2. 从轻重量开始，强调肩胛骨的运动
3. 教练用手按压学员的肩胛骨，让其感受后缩的感觉

#### 膝盖内扣（腿部训练）

**症状**：深蹲或弓步时膝盖向内倾斜\
**快速纠正**：

1. 弹力带绕膝盖，侧向走5-10步激活臀肌
2. 镜子反馈，让学员看到问题
3. 减轻负重，强调膝盖与脚尖对齐

#### 呼吸屏气

**症状**：脸部潮红、屏住气息\
**快速纠正**：

1. 教练示范并带领学员数拍子（吸2，呼2）
2. 强调"难的时候更要呼气"
3. 用手放在腹部感受呼吸运动

#### 营养基础原理

**三大宏量营养物质目标**

| 训练目标 | 热量            | 蛋白质           | 碳水        | 脂肪        |

#### 碳水化合物选择

**优质碳水（推荐）**

- 白米、糙米：全热量来源
- 燕麦：富含纤维，稳定血糖
- 全麦面包：纤维高
- 甘薯、红薯：复合碳水
- 豆类：蛋白+碳水
- 水果（香蕉、苹果、蓝莓）：维生素+纤维

**限制碳水（减脂时避免）**

- 白糖、蜂蜜、果酱
- 白面包、甜点
- 含糖饮料、能量饮料
- 精制零食

**训练周期的碳水调整**

- 训练日：目标碳水量 + 20%（补充能量）
- 非训练日：目标碳水量 - 15%（降低热量）

**推荐脂肪来源**

- 橄榄油：单不饱和脂肪，心血管健康
- 坚果和种子（杏仁、核桃、亚麻籽）：Omega-3脂肪酸
- 鱼油和鱼（三文鱼、沙丁鱼）：EPA/DHA
- 牛油果：单不饱和脂肪
- 椰子油：中链脂肪酸（适量使用）

**限制脂肪来源**

- 反式脂肪（人造黄油、油炸食品）
- 过度饱和脂肪（加工肉类）
