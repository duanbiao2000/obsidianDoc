

1.  **核心级 (Core Class)**
    *   这是目前最小的级别，尺寸较小，变形简单，但细节仍然不错。通常是一些小个子角色或者可以作为大尺寸角色的配件。

2.  **加强级 (Deluxe Class / D级)**
    *   这是最常见和标准的尺寸，也是角色数量最多的一个级别。它在尺寸、细节和变形复杂性之间取得了很好的平衡。

3.  **航行家级 (Voyager Class / V级)**
    *   比加强级更大、更复杂，通常是队伍中的小队长或者体型较大的角色，如擎天柱、威震天等。

4.  **领袖级 (Leader Class / L级)**
    *   体型更大，变形设计更复杂，通常包含更多的配件或变形机关。这个级别的角色通常是阵营的领袖或重要的大型角色。

5.  **指挥官级 (Commander Class)**
    *   一个较大的尺寸级别，介于领袖级和泰坦级之间。通常用于那些体型巨大、在故事中占有重要地位的角色，如天火、补天士等。

6.  **泰坦级 (Titan Class)**
    *   目前官方发行的最大级别，通常是“城市金刚”或组合金刚的巨型角色，例如猛大帅 (Metroplex)、铁甲龙 (Trypticon) 等。


*   **大师级 (Masterpiece / MP):** 这是一个独立的、面向成年收藏家的高端系列。它追求极致的动画或电影造型还原度，变形极其复杂，价格也更高。
