HIIT（高强度间歇训练）核心是“短时间极限发力+短时间休息”交替循环，不用复杂器械，在家就能做，新手也能快速上手，具体执行可按「准备→选动作→定节奏→执行」四步走：

### 一、先做好2个基础准备（避免受伤，提升效果）

1. **热身（5-8分钟，必做）**\
   目的是激活肌肉、提升心率，避免运动中拉伤。\
   推荐动作：高抬腿（30秒）、开合跳（30秒）、弓步压腿（左右各20秒）、手臂绕环（前后各15秒），每个动作间休息10秒，重复1-2轮。
2. **选好运动场地和装备**
   - 场地：空旷的客厅、阳台即可，地面铺瑜伽垫（避免膝盖、手掌受伤）。
   - 装备：穿防滑运动鞋，新手不用负重（徒手即可），有基础可加弹力带、小哑铃。

### 二、新手必学的3套HIIT动作组合（零门槛，燃脂效率高）

HIIT动作以“全身参与、能快速提升心率”为原则，新手优先选简单易标准的动作，避免复杂技巧（如波比跳可先简化）。以下3套组合，任选1套即可，每套15-20分钟，燃脂量相当于慢跑1小时。

#### 组合1：新手入门款（无跳跃，对膝盖友好）

| 动作名称  | 动作要点                     | 发力时间         | 休息时间 |
| ----- | ------------------------ | ------------ | ---- |
| 徒手深蹲  | 双脚与肩同宽，膝盖不超过脚尖，蹲到大腿与地面平行 | 40秒          | 20秒  |
| 跪姿俯卧撑 | 膝盖着地，双手撑地与肩同宽，身体呈直线，屈肘向下 | 40秒          | 20秒  |
| 静态箭步蹲 | 前后腿屈膝90°，前腿膝盖不超过脚尖，保持静态  | 左右各20秒（共40秒） | 20秒  |
| 平板支撑  | 手肘撑地，身体呈直线，核心收紧，不塌腰不翘臀   | 40秒          | 20秒  |

#### 组合2：进阶燃脂款（含低强度跳跃，提升效率）

| 动作名称    | 动作要点                        | 发力时间 | 休息时间 |
| ------- | --------------------------- | ---- | ---- |
| 开合跳     | 双脚跳开与肩同宽，双手举过头顶；跳回并拢，双手放下   | 30秒  | 30秒  |
| 俯身登山跑   | 双手撑地，核心收紧，交替提膝向胸口靠近，速度适中    | 30秒  | 30秒  |
| 臀桥      | 平躺屈膝，双脚踩地，臀部抬起至身体呈直线，顶峰停顿1秒 | 30秒  | 30秒  |
| 高抬腿（原地） | 膝盖尽量抬高至腰腹，脚尖落地，保持上半身稳定      | 30秒  | 30秒  |

#### 组合3：高效冲刺款（适合有1个月基础者）

| 动作名称      | 动作要点                            | 发力时间         | 休息时间 |
| --------- | ------------------------------- | ------------ | ---- |
| 波比跳（简化版）  | 先蹲跳（不用完全跳离地面），再俯身撑地，最后起身（省略俯卧撑） | 20秒          | 40秒  |
| 俄罗斯转体     | 坐姿屈膝，双脚离地，双手合十，左右转体触碰地面         | 20秒          | 40秒  |
| 弓步跳       | 前后腿交替跳，落地时屈膝90°，保持身体稳定          | 20秒          | 40秒  |
| 侧平板支撑（左右） | 单肘撑地，身体呈直线，侧边核心收紧               | 左右各10秒（共20秒） | 40秒  |

### 三、关键执行规则（做好这3点，效果翻倍）

1. **严格控制“发力+休息”节奏**
   - 新手：发力时间≤休息时间（如40秒发力+20秒休息，或30秒发力+30秒休息），避免一开始就过度疲劳。
   - 进阶者：发力时间＞休息时间（如30秒发力+15秒休息），进一步提升心率，强化燃脂效果。
   - 核心原则：发力时“尽全力”（能做到的最快速度、最大幅度），休息时“不瘫坐”（可站立慢走、深呼吸，保持身体轻微活动）。
2. **循环次数和总时长**
   - 每套动作（4个动作）为1轮，新手做3-4轮，进阶者做5-6轮，总时长控制在15-25分钟（超过25分钟易过度消耗，反而影响恢复）。
3. **运动后必做拉伸（5分钟，缓解酸痛）**\
   重点拉伸发力部位：
   - 腿部：大腿前侧（弓步拉伸）、大腿后侧（坐姿体前屈）、小腿（靠墙踮脚拉伸）。
   - 核心：猫式伸展（跪姿，弓背含胸再塌腰抬头）。
   - 手臂：手臂交叉拉伸（一只手举过头顶，另一只手辅助拉向身体）。

### 四、新手常见问题（避坑指南）

1. **跟不上节奏怎么办？**\
   优先保证“动作标准”，再追求速度。比如做深蹲时，若40秒坚持不了，可先做30秒，休息30秒，逐步提升耐力。
2. **多久做一次合适？**\
   HIIT强度高，身体需要恢复，新手每周2-3次（如周一、三、五），每次间隔1天；进阶者每周3-4次，避免每天做导致肌肉拉伤、过度疲劳。
3. **做完头晕、恶心正常吗？**\
   轻微头晕是心率骤升后的正常反应，可放慢呼吸、喝少量温水缓解；若频繁恶心，可能是发力时憋气（需保持呼吸节奏，如发力呼气、放松吸气）或强度过高（可降低发力时间）。

## 周训练计划

以下是一份适合新手的HIIT周计划，结合了低强度、长休息的原则，帮助你逐步适应高强度间歇训练的节奏，同时搭配了一些适合训练的音乐。

### 新手HIIT周计划

- **训练频率**：每周进行2次HIIT训练，每次训练间隔至少1天，给身体足够的恢复时间。
- **训练前准备**：每次训练前进行3分钟热身，如原地高抬腿（2分钟，步频≥80次/分钟）、肩绕环+脚踝环绕（各30秒，幅度逐步加大），让身体做好运动准备，避免受伤。
- **具体训练计划**

| 训练次数 | 动作组合                                                                  | 循环次数 | 训练后拉伸                                           |
| ---- | --------------------------------------------------------------------- | ---- | ----------------------------------------------- |
| 第一次  | “波比跳30秒+休息60秒”“高抬腿30秒+休息60秒”“开合跳30秒+休息60秒”                            | 3组   | 训练后进行10分钟静态拉伸，重点拉伸大腿前侧、腘绳肌、小腿、肩背等部位，每个部位拉伸30秒左右 |
| 第二次  | “深蹲跳30秒，低冲击：落地半蹲缓冲+慢走30秒”“弹力带划船30秒，10磅带+静态拉伸30秒”“俯身登山30秒，膝盖不碰地+慢走30秒” | 3组   | 训练后进行10分钟静态拉伸，重点拉伸训练中用到的肌群，如大腿前侧、小腿、背部等         |

### 训练音乐推荐

- **《狂恋》—万妮达/GALI**：节奏明快，具有较强的动感，能够激发运动的热情，适合在HIIT训练中作为节奏引导音乐。
- **《美丽新世界》—布瑞吉/gosh**：歌曲节奏富有活力，旋律积极向上，能帮助提升训练时的精神状态，让你更投入地进行训练。
- **《Costa Blanca》—Power Sport Team**：140bpm的节奏适中，鼓点清晰，能够为HIIT训练提供稳定的节奏支持，帮助你保持动作的韵律和强度。
- **《Eye of the Tiger》—Power Sport Team**：经典的运动音乐，节奏强烈有力，充满激情和动力，在你进行HIIT训练感到疲惫时，能激励你坚持下去。
