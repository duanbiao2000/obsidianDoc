核心建议：根据你的跨域探索者特质，最终知识库应该长成这样：

脊柱（主题库）：稳定、深度、体系化

神经（问题库）：灵活、快速、跨越多个主题

血管（双向链接）：主题 ← → 问题 ← → 实战案例

这样的知识库才能真正成为你的"外脑"，而不只是信息的堆积。

【主题库速查】(适合定期复习、构建完整知识体系)

主题1: 大脑学习基础机制 | 前置:神经可塑性、工作记忆限制(7±2)
├─ ★ 专注=聚光灯 → 高度集中单一任务
├─ ★ 发散=散光束 → 放松状态，大脑区域跨越
└─ 关键链接: Q4(卡住了咋想解法) → 题4(应试)

主题2: 记忆编码与组块化 | 前置:短期衰减曲线、认知负荷论
├─ ★ 组块=信息打包 → 可作为单元调用
├─ △ 学新概念先分解 → 复杂→可处理的块
└─ 关键链接: Q3(怎么快速掌握新概念) → 题3(应试)

主题3: 对抗'理解的错觉' | 前置:被动阅读局限、遗忘曲线
├─ ★ 主动回忆→破错觉 → 强制提取暴露盲点
├─ ❗ 重读=熟悉≠掌握 → 反复不等于真懂
└─ 关键链接: Q1(学久了还是感觉没掌握) → 题5(检验真假懂)

主题4: 间隔重复与长期保持 | 前置:遗忘曲线、记忆巩固时间窗口
├─ ★ 间隔复习→抗遗忘 → 在遗忘前回顾
├─ △ 复习间隔递增 → 1→3→7→14→30天
└─ 关键链接: Q2(突击复习为啥差) → 题6(做题卡在难题)

主题5: 学习迁移与教授他人 | 前置:迁移类型、教学学习效应
├─ △ 教人→漏洞暴露 → 讲解强制系统化思路
├─ ★ 解释=二次编码 → 用通俗语言转述
└─ 关键链接: Q5(怎样检验真假懂) → 题5(验证方法)

主题6: 应试策略与执行 | 前置:工作记忆与压力、自我效能感
├─ △ 由难而易跳跃 → 从容易题目开始
├─ △ 难题挂起→后台想 → 不纠缠，大脑后台处理
└─ 关键链接: Q6(做题卡在难题) → 题4(困难本质)

主题7: 心态与长期学习 | 前置:固定型vs成长型思维、韧性
├─ ★ 成长思维→能力可塑 → 能力非固定
├─ ★ 毅力→马拉松学习 → 长期坚持>短期冲刺
└─ 关键链接: Q7(学很久还感觉无进步) → 题7(困难应对)

主题8: 团队学习与知识沉淀 | 前置:集体认知、心理安全、显性化知识
├─ △ 团队学习→心理安全 → 鼓励提问犯错
├─ △ 知识文档化→可复用 → 隐性变显性
└─ 关键链接: Q8(团队知识传递) → 题8(项目协作)

【问题库速查】(适合遇到具体问题、实战演练)

Q1: 为什么学了很久还是感觉没掌握？
├─ 关键概念: 理解的错觉 / 主动回忆 / 被动vs主动
├─ 对抗策略: ★主动回忆→破错觉 / △卡片系统 / △口述解题
└─ 实战: 做题→Anki→教人

Q2: 临考前突击复习为什么效果差？
├─ 关键概念: 遗忘曲线 / 间隔效应 / 长期记忆编码
├─ 核心: ★间隔复习→抗遗忘 / △复习间隔递增 / ❗突击短效
└─ 实战: 提前2-4周→每周间隔→考前补强

Q3: 怎么才能快速掌握一个复杂的新概念？
├─ 关键概念: 认知负荷 / 组块化 / 先大图后细节
├─ 核心: ★组块=信息打包 / △学新概念先分解
└─ 实战: 总览→拆分→逐块学→整体复述

Q4: 卡住了，怎样才能想出解法？
├─ 关键概念: 两种大脑模式 / 发散模式 / 后台处理
├─ 核心: ★卡住→两模式交替 / ❗新问题靠发散
└─ 实战: 专注15分钟→没进展时休息→散步思考→回来新视角

Q5: 怎样检验自己是真懂还是假懂？
├─ 关键概念: 主动提取 / 教学效应 / 盲点暴露
├─ 核心: ★主动回忆→破错觉 / △教人→漏洞暴露
└─ 实战: 合上书讲→给别人讲→做变式题

Q6: 做题时老是卡在难题上，怎么办？
├─ 关键概念: 工作记忆资源 / 自我效能感 / 信心积累
├─ 核心: △由难而易跳跃 / △先拿稳分→信心↑
└─ 实战: 扫全卷→先做确信题→难题空着→后回头

Q7: 为什么有时学得很努力还是感觉进步不了？
├─ 关键概念: 成长思维 / 毅力 / 策略vs蛮力
├─ 核心: ★成长思维→能力可塑 / ★毅力→马拉松
└─ 实战: 确认策略→每天坚持→记录进展→看复利

Q8: 在团队项目中，怎样保证知识能被复用和传递？
├─ 关键概念: 知识沉淀 / 显性化 / 心理安全 / 元沟通
├─ 核心: △知识文档化 / △代码评审 / △同行评审
└─ 实战: 写文档→知识分享→新人清单→定期复盘

【使用流程】

1️⃣ 初次学习 → 按主题顺序深学 (1-2周)
   主题1 → 主题2 → ... → 主题8

2️⃣ 实战演练 → 遇到问题用问题库快查 (2-8周)
   遇问题 → 查Q1-Q8 → 找对应主题深化 → 执行实战方案

3️⃣ 定期复习 → 用主题库完整复盘 (持续)
   每2周 → 按主题顺序复述 → 检查理解深度 → 发现不足

4️⃣ 建立链接 → 在笔记工具中双向关联 (第2-4周开始)
   主题页 ← 链向相关问题
   问题页 ← 链向对应主题和实战案例
