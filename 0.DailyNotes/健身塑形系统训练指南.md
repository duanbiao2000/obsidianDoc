# 健身塑形系统训练指南

### 1. 核心架构与法则

- **4D 矩阵**：强度（X）、器械（Y）、能力（Z）、目标（时间）。
- **呼吸力学**：腹式（稳定/恢复）、胸式（力量）、节律（有氧）。
- **动作流程**：离心控制（2-3s）、顶峰收缩（1s）、向心爆发（1s）。
- **渐进超载**：应力 $\to$ 适应 $\to$ 超补偿。通过增加重量、次数或缩短间歇实现。
- **肌肉混淆**：每 2-6 周调整动作角度、器械、节奏或系统。
- **功能模式**：深蹲、铰链、弓步、推、拉、旋转、负重行走。

---

### 2. 目标执行策略

- **减脂塑形**
  - **核心**：热量缺口 + 力量训练 + HIIT/LISS。
  - **比例**：高蛋白（$2.0\text{g/kg}$），热量缺口 $300\text{-}500\text{kcal}$。
- **增肌塑形**
  - **核心**：热量盈余 + 肌肥大区间（8-12 次/组）。
  - **比例**：高碳水（$5\text{-}6\text{g/kg}$），热量盈余 $200\text{-}400\text{kcal}$。
- **运动表现**
  - **核心**：力量-速度曲线全谱训练（1-5RM 力量 $\to$ 爆发力 $\to$ 耐力）。

---

### 3. 核心功能分层

| 维度 | 训练重点 | 典型动作 | |
| --- | --- | --- | --- |
| **抗伸展** | 核心前侧稳定 | 平板支撑、死虫式 | |
| **抗旋转** | 核心扭转稳定 | 帕洛夫推举、单臂行走 | |
| **抗侧屈** | 侧向稳定 | 侧平板、单侧负重行走 | |

---

### 4. 营养与恢复

- **热量计算**：$\text{BMR} \to \text{TDEE} \to \text{目标热量}$。
- **必需补剂**：乳清蛋白（修复）、肌酸（力量）、鱼油（抗炎）。
- **恢复金字塔**：睡眠（基础） $>$ 营养 $>$ 拉伸/泡沫轴 $>$ 理疗。
