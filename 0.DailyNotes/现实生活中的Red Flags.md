# 现实生活中的 Red Flags（危险信号）

---


## 人际关系

### 恋爱/婚姻

| Red Flag | 信号 |
|----------|------|
| "我这都是为你好" | 控制欲的伪装 |
| "如果你爱我，你就会..." | 情感勒索 |
| "我的前任都是疯子" | 问题可能在 TA 自己 |
| 过早说"我爱你" | Love bombing，可能是 PUA |
| 从不道歉 / 永远是别人的错 | 缺乏责任感 |
| 孤立你与朋友家人的关系 | 控制的前兆 |
| 查手机、行踪 | 不信任 / 控制欲 |

### 友谊

| Red Flag | 信号 |
|----------|------|
| 只在需要时联系你 | 工具人友谊 |
| 当面一套背后一套 | 不值得信任 |
| 总是贬低你的成就 | 嫉妒心 |
| 分享秘密后被传播 | 无法保守信任 |

## 职场

| Red Flag | 信号 |
|----------|------|
| "我们是一家人" | 画饼、模糊边界、无偿加班 |
| "年轻人要多吃苦" | 剥削的借口 |
| 面试时诋毁前员工 | 管理有问题 |
| 高离职率 | 内部肯定有问题 |
| "弹性工作制" | 可能意味着 007 |
| 入职要求交钱/押金 | 大概率是骗局 |
| 职责模糊不清 | 什么都让你干 |

| Red Flag | 信号 |
|----------|------|
| "稳赚不赔" | 没有这种事 |

## 通用原则

> **"如果听起来好得不像真的，那它很可能就不是真的。"**
> — If it sounds too good to be true, it probably is.


---

## 识别 Red Flags 的心法

| 原则 | 解释 |
|------|------|
| 🔴 紧迫感 | 催你快做决定 → 不让你思考 |

| 🔴 秘密性 | 让你不要告诉别人 → 怕曝光 |
| 🔴 情感绑架 | 用愧疚/恐惧操控你 |
| 🔴 信息不对等 | 你不懂，我来帮你 → 容易被骗 |
| 🔴 过度承诺 | 画饼太大 → 无法兑现 |

---

保持警惕，但也不必草木皆兵。关键是**培养判断力**，遇事多问：
- 对方的动机是什么？
- 最坏的情况是什么？

- 我有什么可以核实的渠道？
