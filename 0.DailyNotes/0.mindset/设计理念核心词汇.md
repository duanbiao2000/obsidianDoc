这是一个从你的 Prompt 中提取的高价值设计词汇表，按**设计维度**进行了分类。

这些词汇不仅适用于生成这个网站，也是描述**现代主义（Modernism）** 和 **瑞士平面设计风格（Swiss Style）** 的核心术语。

### 1. 核心风格与理念 (Core Style & Philosophy)

描述整体的视觉调性和设计哲学。

|**英文 (English)**|**中文 (Chinese)**|**设计含义**|
|---|---|---|
|**Swiss-Style / International Typographic Style**|**瑞士风格 / 国际主义平面设计风格**|强调清洁、可读性和客观性的设计流派。|
|**Ultra-Minimalist**|**极致极简**|去除所有非必要元素，只保留最核心的内容。|
|**Reductive**|**做减法的 / 克制的**|通过减少元素来增强表达力（Less is More）。|
|**Functional Visual Language**|**功能性视觉语言**|每一个视觉元素都必须有其实际用途，而非单纯装饰。|
|**Honesty / Clarity**|**诚实 / 清晰**|设计不掩盖内容的本质，直观传达信息。|
|**No Fluff**|**无冗余 / 不花哨**|拒绝毫无意义的装饰和废话。|

### 2. 布局与网格系统 (Layout & Structure)

瑞士风格的灵魂在于网格和秩序。

|**英文 (English)**|**中文 (Chinese)**|**设计含义**|
|---|---|---|
|**Grid-Driven**|**网格驱动**|所有元素严格遵循底层网格对齐。|
|**Rigid Baseline Grid**|**严格的基线网格**|文字行高和间距遵循统一的垂直节奏。|
|**Modular System**|**模块化系统**|像积木一样可组合、可替换的布局单元。|
|**Asymmetrical Layouts**|**不对称布局**|也就是“非对称平衡”，通过留白和大小对比而非居中对称来创造动态平衡。|
|**Precision Spacing**|**精准间距**|间距（Padding/Margin）遵循严格的数学倍数（如 8px, 16px）。|
|**Generous Whitespace / Breathe**|**大量留白 / 呼吸感**|使用负空间来突出主体，防止视觉拥挤。|

### 3. 排版与文字 (Typography)

文字即图像，排版即设计。

|**英文 (English)**|**中文 (Chinese)**|**设计含义**|
|---|---|---|
|**Grotesk Typeface**|**哥特类无衬线体**|如 Helvetica, Inter 等，线条均匀，冷静客观。|
|**Bold Typographic Hierarchy**|**大胆的排版层级**|通过极端的字号对比（极大标题 vs 极小正文）来引导视线。|
|**Tight Line Height**|**紧凑行高**|标题行距很小，营造块面感和冲击力。|
|**Declarative**|**宣言式的**|文本短促、有力，像是在发表声明。|
|**Monospace**|**等宽字体**|常用于辅助信息或代码感，增加一种“数据化/机械化”的美感。|

### 4. 视觉元素与细节 (Visual Elements & Details)

具体的图形处理方式。

|**英文 (English)**|**中文 (Chinese)**|**设计含义**|
|---|---|---|
|**High-Impact**|**高视觉冲击力**|第一眼就能抓住注意力的设计强度。|
|**Full-Bleed**|**满版 / 满出血**|图片或色块铺满整个屏幕宽度，无边框。|
|**Geometric Accents**|**几何装饰元素**|使用简单的线条、矩形、圆形作为点缀。|
|**No Gradients / No Shadows**|**无渐变 / 无投影**|坚持扁平化（Flat），拒绝模拟现实的立体感。|
|**Invert on Hover**|**悬停反色**|鼠标经过时黑白互换，一种经典的高对比交互反馈。|

### 💡 如何使用这些词汇？

- **当你要修改设计时：** 可以对 AI 说 "The spacing feels too tight, let's add more **generous whitespace** to let the content **breathe**."（间距太紧了，增加一些**大量留白**让内容**呼吸**。）
    
- **当你定义品牌时：** 可以说 "We want a **reductive**, **functional visual language** emphasizing **clarity**."（我们需要一种**做减法的**、**功能性的视觉语言**，强调**清晰度**。）
