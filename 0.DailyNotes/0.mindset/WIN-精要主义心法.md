
**一句话价值主张**：提供一套系統性的思維與行動框架，用於辨識、排除並高效執行真正重要的少數事務，實現「更少，但更好」的個人與職業產出。

**立即行动**：
1.  **評估機會時**：啟動「90%法則」，若一個選項的重要性評分低於90分，則直接拒絕。
2.  **感到精力分散時**：制定一個清晰、鼓舞人心的「精要目標」，用它來過濾所有不相關的活動。
3.  **進度停滯時**：找出阻礙你前進的唯一「最慢行進者」（瓶頸），集中資源將其掃除。

**Cheat Sheet**：

| 触发场景 | 精准命令/公式 | 验证方法 |
|---|---|---|
| 面對過多機會或請求 | 應用 **90%法則** 篩選 → 答案非「絕對是」即為「否」 | 待辦清單只剩下極少數高價值選項 |
| 感覺忙碌但無成就感 | 澄清個人/團隊的 **精要目標** (Essential Intent) | 能用一句話說出當前最重要的事，並據此決策 |
| 專案進展緩慢 | 識別並掃除唯一的**瓶頸** (Constraint) | 移除障礙後，整體進度顯著提升 |
| 害怕拒絕他人 | 將「決定」與「關係」分離 → 優雅地說「不」 | 保護了個人時間，且未損害長期人際關係 |
| 因過去的投入而猶豫 | 忽略**沉沒成本** → 問：「若尚未投入，我現在會投嗎？」 | 成功從無價值的承諾中抽身，釋放資源 |
| 執行重要任務時分心 | 專注當下 (WIN: What's Important Now?) → 進入心流 | 任務完成質量和效率顯著提高 |

*   **重构**：
    *   `[混乱逻辑]`：筆記結構龐大，分為8個問答、4個Part和一個關鍵詞彙表，存在大量重複。建議重構為一個統一的 **「思維模式 → 行動框架 → 應用場景」** 的三層結構：
        1.  **Mindset (核心思維)**：選擇 (Choose)、甄別 (Discern)、取捨 (Trade-Off)。這是所有行動的基礎。
        2.  **Framework (三步框架)**：
            *   **Explore (探索)**：系統性地尋找那「極少數重要的事」。
            *   **Eliminate (排除)**：堅決地砍掉那「大多數不重要的事」。
            *   **Execute (執行)**：設計體系，讓「重要的事」執行起來毫不費力。
        3.  **Application (應用場景)**：將框架應用於個人生活 (Personal) 和團隊領導 (Leadership) 兩個領域。
