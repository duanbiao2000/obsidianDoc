
# 认知升维：可重复的反思笔记（反思 + 行动）

## 一、今日反思（3问）
- 我今天产出的“知识”里，有哪一条能跨域复用？为什么？
- 哪个观点可以被压缩成“关系图/规则”，并指导明天的决策？
- 我给这条“洞见”设定了边界与反例了吗？如果没有，它仍是观点而非模型。

## 二、升维判据（成功才收口）
- 迁移成功≥1次（跨不同场景）  
- 压缩率≥50%（笔记字数/节点下降）  
- 预测命中≥60%（对新样本的建议有效）  
- 复现≤30min（他人按卡片复现要点）

## 三、30分钟微SOP（从信息到模型）
1) 收敛样本（5–8min）  
   - 选3–5个代表案例；记录“输入-过程-输出-环境”。
2) 提炼关系（8–10min）  
   - 写3条“因-果-约束”；画极简关系图（≤7节点）。
3) 边界与反例（5–7min）  
   - 适用边界×3；反例×1（明确破坏点）。
4) 脚手架化（5–7min）  
   - 什么时候用/不用；三步可执行流程；停机阈值（Gate）。
5) 快速迁移（≤5min）  
   - 在异域小样本上试用一次，记录命中点与失败点。

> Gate：未达“迁移≥1、命中≥60%、复现≤30min”则回退补样本或缩粒度。
