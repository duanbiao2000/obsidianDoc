智力不是单一的分数（IQ），是三个独立子系统的平衡。

### 核心三元

| 智力类型 | 核心功能 |
| --- | --- |
| **分析性** | 判断、比较、解题（学术能力） |
| **创造性** | 创新、处理新异情境 |
| **实践性** | 适应、塑造环境（生存智慧） |

- **独立性**：分析能力强不代表实践能力强（高分低能）。
- **局限性**：传统 IQ 测试仅测量分析性智力，忽视创造与实践。
- **成功智力**：在于三种能力的平衡运用。

### 情境与文化

智力的定义取决于文化背景。

- **西方标准**：逻辑、说服、质疑传统。
- **克里人（Cree）标准**：尊重长辈、社会和谐、生存技能。
- **冲突**：在一种文化中被视为“智慧”的行为，在另一种文化中可能被视为“愚蠢”或“倒退”。

### 结论

- 智力是多维的。
- 成功的关键在于：根据目标，在特定情境下调用匹配的能力。
