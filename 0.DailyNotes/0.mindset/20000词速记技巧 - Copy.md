# AI 时代高效词汇策略（F/C/P/M 升维版）

## F｜事实清单（可检索）
- 术语：词频（Collins/真题）、错题本、记忆锚（谐音/结构/语义）、词根词缀、应用场景（读/写/题）
- 工具与来源：真题导出错词 → 去重 → 标星；Collins 五星；个人错题本
- 阈值初设：加入词库条件 = 词频≥中 + 个人错率≥2 次/周

## C｜概念模型（为什么）
- 记忆效率 ≈ 锚点强度 × 复现频率 × 语境多样性
- 最小必要词库：以战养战（错题数据优先） > 盲目扩容
- 迁移路径：词根结构化 → 语境迁移（读/写/题）→ 策略得分（排除/情感/词性）

## P｜流程（可复现）
- 词条纳入决策（每日）
  1) 数据：当日错题词汇清单
  2) 判定：若 词频≥中 且 本周错≥2 → 纳入；否则暂存观察
  3) 去歧义：选 1 义项 + 1 代表搭配
- 锚点构建（单词≤90 秒）
  1) 锚定：谐音/结构/语义三选一（唯一且不混淆）
  2) 例句：1 条可复述例句；1 个常见搭配
  3) 关联：词根/同族 1 个
- 复习与应用（间隔重复）
  - D0/D2/D7/D21 复习；每次需：回忆→口述例句→造句一次
  - 若回忆<80%：改为更强锚点 + 加一天补点
- 场景迁移（每周≥2 次）
  - 阅读：标注同根词 3 个
  - 写作：本周词库中至少 5 个用于段落

## M｜目标-监控-调整（闭环）
- 目标（周）：
  - 纳入新词 ≤ 40；稳定保留率 ≥ 85%
  - 错题重复率 ≤ 上周的 70%
  - 写作实际使用 ≥ 15 次（独立词形）
- 监控（每日 21:30）：
  - 新/复习/使用 数量；回忆正确率；错题复发数
- 调整（触发器与动作）：
  - 若保留率<80% → 降低新词上限至 20，并统一改“结构锚点”；
  - 若写作使用<10 次 → 次日读写各加入“必用清单”5 词；
  - 若错题复发>5 → 本周仅做“错题族群精修”，暂停新域扩张
