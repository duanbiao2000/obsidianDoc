# 大脑逻辑-决策-执行串联机制 (Brutally Minimal)

## 核心架构

**高度互联神经网络 + 持续反馈循环**

类比：CEO(前额叶)协调下的组织，部门(脑区)不断沟通、协作、执行、调整

---

## 主要脑区

**逻辑与推理**
- 背外侧前额叶皮层 (dlPFC): 工作记忆、抽象思维、问题解决、规划、逻辑推理
- 下顶小叶: 数字推理、空间推理、抽象概念整合

**决策**
- 腹内侧前额叶/眼眶额叶 (vmPFC/OFC): 整合情绪、价值、奖赏信息，评估后果/风险/收益
- 前扣带皮层 (ACC): 冲突监测、错误检测、认知负荷评估
- 杏仁核: 情绪处理(恐惧/焦虑)，影响风险评估
- 伏隔核 + 多巴胺系统: 奖赏通路，影响价值判断

**执行**
- 前额叶 (PFC): 决策→行动计划，组织动作序列
- 前运动/辅助运动区: 高层次运动规划和协调
- 初级运动皮层 (M1): 向肌肉发送指令
- 基底神经节: "门控"选择器，决定哪个运动计划执行，抑制无关运动
- 小脑: 运动协调、精确性、平衡，实时微调

---

## 串联流程

**并行 + 迭代 + 皮层-皮层下-皮层环路**

### 1. 信息整合与问题构建

```
感觉输入(感官皮层) + 记忆(海马体)
         ↓
      dlPFC
整合信息 → 工作记忆 → 问题表征 → 识别目标 → 生成潜在方案
```

### 2. 逻辑推理与方案生成

```
dlPFC
抽象分析 → 逻辑规则 → 模式识别 → 推断
         ↓
生成多个可能方案(头脑沙盘推演)
```

### 3. 价值评估与决策选择

```
vmPFC/OFC
接收dlPFC方案 + 杏仁核(情绪) + 伏隔核(奖赏)
         ↓
评估: 潜在奖赏 + 风险 + 情感后果 + 价值观符合度
         ↓
      ACC
监测不确定性/冲突 → 向dlPFC反馈 → 更深推理/寻找替代
         ↓
反馈回路: 过去经验 + 情感/奖赏结果 → 影响当前评估
```

### 4. 行动规划

```
决策敲定
    ↓
dlPFC
抽象决策 → 具体行动计划
    ↓
分解任务 + 排序 + 资源分配
    ↓
前运动/辅助运动区
准备组织运动序列(未执行)
```

### 5. 运动执行与监测

```
基底神经节-丘脑-皮层环路
接收计划 → 筛选 → 抑制无关动作 → 释放选定计划
         ↓
    初级运动皮层
接收输出 → 向肌肉发送指令
         ↓
      小脑
持续接收感觉反馈(位置/平衡/视觉)
比较预期与实际 → 实时调整 → 确保精确/流畅/平衡
```

### 6. 结果反馈与学习(闭环)

```
行动执行
    ↓
感官反馈(视觉/听觉/触觉/本体感觉)
    ↓
奖赏信号(多巴胺) / 错误信号(ACC)
    ↓
反馈送回 PFC(OFC/vmPFC) + 相关区域
    ↓
更新世界理解 + 修正决策模型 + 强化成功策略 + 调整失败策略
    ↓
影响未来逻辑推理和决策(学习与适应基础)
```

## 关键特征

**非线性**
高度并行、迭代、相互连接

**环路机制**
皮层-皮层下-皮层环路

**持续反馈**
感觉 → 执行 → 结果 → 更新 → 下次循环

**核心枢纽**
- 前额叶: 执行中枢，高级协调整合
- 基底神经节: 意图 → 精确行动门控
- 小脑: 实时微调运动精确性

**学习优化**
反馈机制持续调整思考和行为

## 简化模型

```
输入(感觉+记忆)
    ↓
dlPFC: 逻辑推理 → 方案生成
    ↓
vmPFC/OFC: 价值评估 → 决策选择
    ↓
ACC: 冲突监测 → 反馈优化
    ↓
PFC: 行动规划 → 任务分解
    ↓
基底神经节: 门控选择 → 释放计划
    ↓
M1: 发送指令 → 肌肉执行
    ↓
小脑: 实时微调 → 精确协调
    ↓
结果反馈 → 学习更新 → 影响下次循环
```
