“Agent Framework”（智能体框架）是指用于构建、训练、部署和管理 AI 智能体（Agent）的软件框架或平台。这些框架通常提供模块化组件，如记忆（Memory）、工具调用（Tool Use）、规划（Planning）、推理（Reasoning）、多智能体协作（Multi-Agent）等功能。

你提到的 **Mastra** 是一个新兴的开源智能体框架，下面我会先简要介绍 Mastra，然后列出当前主流或值得关注的 Agent Framework。

---

## 1. Mastra

- **简介**：Mastra 是一个面向开发者的开源智能体框架，强调模块化、可扩展性和易用性。它支持多智能体协作、工具集成、长期记忆、自主任务执行等。
- **特点**：
    - 支持多种 LLM（如 OpenAI、Anthropic、Ollama 等）
    - 内置工具调用（Tool Calling）和函数注册机制
    - 提供 Web UI 和 CLI
    - 支持本地部署
- **GitHub**：[https://github.com/mastra-ai/mastra](https://github.com/mastra-ai/mastra) （截至 2025 年较活跃）

## 2. 主流 Agent Framework 列表（截至 2025 年）

### 🔹 LangChain

- **定位**：最广泛使用的 LLM 应用开发框架之一，也支持构建简单 Agent。
- **特点**：丰富的工具集成、文档完善、社区庞大。
- **局限**：早期设计偏向链式调用（Chains），Agent 能力在不断演进中。
- **官网**：[https://www.langchain.com/](https://www.langchain.com/)

### 🔹 LlamaIndex（原 GPT Index）

- **定位**：专注于数据连接与检索增强生成（RAG），但也支持轻量级 Agent。
- **特点**：与 LangChain 互补，适合知识密集型 Agent。
- **官网**：[https://www.llamaindex.ai/](https://www.llamaindex.ai/)

### 🔹 AutoGen（Microsoft）

- **定位**：多智能体对话框架，由微软研究院推出。
- **特点**：
    - 支持多个可定制 Agent 协作（如 coder、executor、reviewer）
    - 可结合人类反馈（human-in-the-loop）
    - 支持本地/远程执行
- **GitHub**：[https://github.com/microsoft/autogen](https://github.com/microsoft/autogen)

### 🔹 CrewAI

- **定位**：受 AutoGen 启发，更强调“角色扮演”和任务导向的多智能体系统。
- **特点**：
    - 简洁 API，易于定义角色（Role）、目标（Goal）、工具（Tools）
    - 支持分步任务编排（Task Delegation）
- **官网**：[https://www.crewai.com/](https://www.crewai.com/)
- **GitHub**：[https://github.com/joaomdmoura/crewAI](https://github.com/joaomdmoura/crewAI)

### 🔹 Semantic Kernel（Microsoft）

- **定位**：轻量级 SDK，用于将 LLM 与传统代码融合。
- **特点**：插件化设计，支持 .NET、Python、Java。
- **适合场景**：企业级应用集成，非纯 Agent 导向但可构建 Agent。
- **官网**：[https://learn.microsoft.com/en-us/semantic-kernel/](https://learn.microsoft.com/en-us/semantic-kernel/)

### 🔹 Haystack（deepset）

- **定位**：NLP pipeline 框架，主要用于问答和 RAG，也支持 Agent-like 流程。
- **特点**：模块化 pipeline，支持自定义节点。
- **官网**：[https://haystack.deepset.ai/](https://haystack.deepset.ai/)

### 🔹 Transformers Agents（Hugging Face）

- **定位**：Hugging Face 提供的实验性 Agent 接口。
- **特点**：直接调用 Hugging Face 模型和工具，适合研究。
- **注意**：目前处于实验阶段，生产使用需谨慎。
- **文档**：[https://huggingface.co/docs/transformers/transformers_agents](https://huggingface.co/docs/transformers/transformers_agents)

### 🔹 AgentScope（by Alibaba Tongyi Lab）

- **定位**：阿里通义实验室推出的多智能体开发框架。
- **特点**：
    - 支持复杂多智能体交互协议
    - 高性能消息传递机制
    - 与 Qwen 大模型深度集成
- **GitHub**：[https://github.com/alibaba/agentscope](https://github.com/alibaba/agentscope)

### 🔹 DSPy（Stanford）

- **定位**：不是传统 Agent 框架，而是“声明式语言模型编程”框架。
- **特点**：通过优化提示和模块组合自动提升性能，可用于构建可靠 Agent pipeline。
- **官网**：[https://dspy.ai/](https://dspy.ai/)

### 🔹 Flowise / Langflow

- **定位**：可视化 Agent 编排工具（基于 LangChain）。
- **特点**：拖拽式界面，适合快速原型。
- **适合人群**：非开发者或低代码用户。

## 如何选择 Agent Framework？

|需求|推荐框架|
|---|---|
|快速上手、丰富生态|LangChain + LangGraph|
|多智能体协作|AutoGen、CrewAI、AgentScope|
|企业集成|Semantic Kernel|
|RAG + Agent|LlamaIndex + LangChain|
|可视化编排|Flowise / Langflow|
|轻量 & 实验|Mastra、Transformers Agents|
|声明式优化|DSPy|
