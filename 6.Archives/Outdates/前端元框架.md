### **五、 特定功能庫 (Utility Libraries)**

#### **TanStack：功能強大的“無頭”工具集**

37. TanStack 是一個包含一系列與框架無關的數據管理和 UI 庫的項目。
38. 其中最著名的是 TanStack Query（原 React Query），一個強大的數據獲取和緩存庫。
39. TanStack Table 則是一個用於構建高性能、可定制數據表格的“無頭 UI”庫。
40. TanStack Router 是一個新興的、類型安全的客戶端路由庫。
41. 這些庫功能強大且高度模塊化，可以被集成到任何前端項目中。
42. 它們的設計理念是“與框架無關”，但目前在 React 生態中最為流行。

#### **React Router：React 路由的標準方案**

43. React Router 是一個專門為 React 應用設計的客戶端路由庫。
44. 它的唯一職責是管理單頁應用中的 URL 與 UI 組件的同步。
45. 作為 React 生態中最常用的路由庫，它擁有豐富的功能和活躍的社區。
46. 它是構建幾乎所有 React 單頁應用的基礎。
47. 與 TanStack 相比，React Router 的用途非常單一和專注。
48. 而 TanStack 旗下的 TanStack Router 則是 React Router 的一個直接競爭者。
