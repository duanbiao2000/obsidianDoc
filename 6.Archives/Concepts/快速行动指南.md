# 消除不确定性：快速行动指南

> 💡 **唯一目标：找到可立即执行的第一步**

---

## 核心问题

**不确定 → 大脑逃避 → 拖延瘫痪**

---

## 七种迷雾（快速诊断）

| 类型 | 症状 | 快速检测问题 |
|:---:|---|---|
| **目标** | 不知道要什么结果 | "成功长什么样？" |
| **信息** | 缺关键数据 | "缺什么信息无法决策？" |
| **依赖** | 不知道先做啥 | "这件事依赖什么前置条件？" |
| **范围** | 不知道要做多少 | "边界在哪？什么不做？" |
| **风险** | 担心会出事 | "最坏情况是什么？" |
| **方法** | 不知道怎么做 | "用什么工具/流程？" |
| **认知** | 脑子一团乱 | "能一句话说清吗？" |

---

## 4步破局工作流

### Step 1: 倾倒（2分钟）
**把脑子掏空到纸上**

```
问题：[用一句话描述]
脑子里的乱麻：
- [想法1]
- [担心1]
- [疑问1]
...
```

---

### Step 2: 诊断（3分钟）
**勾选迷雾类型**

```
□ 目标不清  □ 缺信息  □ 不知依赖  □ 范围模糊
□ 担心风险  □ 没方法  □ 认知混乱
```

**选最致命的1-2个**

---

### Step 3: 拆解（5分钟）
**针对迷雾提问**

| 迷雾 | 破局问题 | 答案 |
|---|---|---|
| 目标 | 最小可验证成果是什么？ | [X] |
| 信息 | 缺哪个关键数据？谁有？ | [Y] |
| 依赖 | 必须先完成什么？ | [Z] |
| 范围 | 什么坚决不做？ | [W] |

---

### Step 4: 锁定第一步（1分钟）

**填空题：**
```
我的下一步行动是：
[动词] + [具体对象] + [时间/地点]

示例：
❌ "研究方案"（太模糊）
✅ "今天下午3点给张三发消息问ABC数据"
```

---

## 快速决策模板

### 模板A：缺信息型
```
问题：[X]
缺什么：[具体信息]
谁有：[人名/渠道]
第一步：今天[时间]联系[谁]问[什么]
```

### 模板B：目标模糊型
```
模糊目标：[大而化之的描述]
最小成果：[可在1周内验证的具体结果]
验证标准：[3个可观察指标]
第一步：[制作最小原型的动作]
```

### 模板C：方法未知型
```
要做：[任务]
不知道：[具体卡点]
学习源：[教程/案例/专家]
第一步：花30分钟看[具体资源]做笔记
```

## 实战案例

### ❌ 瘫痪状态
> "我要做个副业，但不知道从哪开始..."

### ✅ 破局过程

**Step 1 倾倒**
```
- 想做内容创作赚钱
- 不知道做什么平台
- 担心没人看
- 不知道怎么涨粉
```

**Step 2 诊断**
```
☑ 目标不清（什么叫"成功"？）
☑ 方法未知（具体怎么做？）
□ 其他
```

**Step 3 拆解**
| 迷雾 | 答案 |
|---|---|
| 目标 | 最小目标：发3篇内容，获得10个真实反馈 |
| 方法 | 先看3个对标账号的前10篇内容 |

**Step 4 锁定**
```
第一步：今晚8点在小红书搜索"副业"，
截图点赞前3的内容，分析选题规律
```

## 关键原则

| 原则 | 说明 |
|:---:|---|
| **具体胜过完美** | "发消息问数据" > "深入研究" |
| **1个行动 > 10个想法** | 做了才知道对不对 |
| **时间锁定** | 没有时间的计划 = 幻想 |
| **动词开头** | 打电话/写文档/测试... |

---

## 紧急救援卡

**当你又卡住时：**

```
1. 设定5分钟倒计时
2. 问自己："如果只能做1件事推进，是什么？"
3. 把答案写成：[动词]+[对象]+[时间]
4. 立刻去做
```

---

✅ **现在就做**：拿出一件拖延的事，用Step 1-4走一遍（总计11分钟）
