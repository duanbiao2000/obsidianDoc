## 语言能力日练计划（基于[[衡量参与者语言能力]]）

---

### 核心原则

- 输入/输出平衡：听/读 ↔ 说/写  
- 刻意练习：对准短板  
- 持续反馈：自检/他评  
- 情境应用：真实场景用起来  
- 每日坚持：30–60 min

---

## 每日框架（30–60 min）

### 模块1 听力（理解）

目标：信息捕捉 / 口音语速适应 / 含义推断

每日选 1–2 项（10–15 min）：

- 精听：1–3 min 新闻/播客片段 → 反复听+听写 → 对照原文改错（连读/弱读/语调）  
- 泛听：长音频/视频 → 抓主旨/关键信息，不查每个词  
- 口音/语速：刻意换地区/语速材料

---

### 模块2 口语（表达）

目标：流利度 / 发音 / 语法 / 词汇 / 语用

每日选 1–2 项（10–15 min）：

- 跟读/Shadowing：跟母语者音频同步模仿（节奏/语调）  
- 朗读：录音→回放→纠正发音/停顿  
- 主题口述/复述：1–2 min 讲当天听/读内容，刻意用新词新句；录音自评  
- 场景对话：问路/点餐/讨论… 自演对话，注意礼貌/语境合适

---

### 模块3 阅读（理解）

目标：速度 / 准确 / 结构感 / 信息抓取

每日选 1–2 项（10–15 min）：

- 精读：短文（评论/摘要）→ 拆句子结构/用词 → 段落要点  
- 泛读：新闻/博客/小说章节 → 快读抓主旨，不停查词  
- 批判性阅读：观点类文章 → 看论点/论据/偏见 → 想自己立场

---

### 模块4 写作（表达）

目标：语法 / 词汇 / 篇章结构 / 论证清晰

每日选 1–2 项（10–15 min）：

- 日记/自由写：随便写一段，重在持续输出  
- 摘要/评论：写今天读/听内容的总结或观点  
- 语法/句型仿写：学一条语法/句型 → 多造句  
- 翻译：短母语段落 → 目标语，追求自然/准确

---

### 模块5 基础能力（词汇/语法/语用）

目标：词汇量 / 核心语法 / 语用感

可融入以上或单独 5–10 min：

- 词汇：Anki/词书/读听中捞词 → 记搭配+例句  
- 语法：定期过核心语法 → 练题 + 在口语/写作中刻意用  
- 语用：观察母语者表达请求/拒绝/道歉/建议 → 模仿句式/语气

---

## 每周整合与反馈

- 综合任务：  
  - 例：听一场讲座 → 写总结 + 口头复述；  
  - 或：读一文 → 写评论 + 录音讨论  
- 自评：  
  - 对照[[衡量参与者语言能力]]各维度：听/说/读/写 + 词汇/语法/语用 → 标出本周进步/短板  
- 外部反馈：  
  - 把录音/作文给老师/语伴/母语者 → 要具体建议

---

## 个性化调整

1. 诊断：  
   - 模拟测试 / 自评 / 对照评估维度 → 找短板  
2. 聚焦：  
   - 把更多时间投到短板模块（如口语弱 → 多 Shadowing + 主题口述）  
3. 难度调节：  
   - 保持在「略吃力但能懂大部分」区间  
4. 兴趣优先：  
   - 素材尽量选自己喜欢的主题，提高可持续性
