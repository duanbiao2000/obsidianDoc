# [[Prompt Pattern]] | Minimal

---

## 0. 本质 (The Essence)
- **定义**：Prompt Pattern = 认知脚手架。
- **逻辑**：将专家的思维路径固化为结构化指令，引导 AI 协助完成高质量的认知加工。
- **定位**：不是简单的问答，而是通过“过程控制”生成深度的“知识节点”。

---

## 1. 核心十式速查

| 模式 | 核心逻辑 | 典型产出 |
| :--- | :--- | :--- |
| **深度解析** | 定义 + 类比 + 要素 + 误区 | 陌生概念的系统理解 |
| **魔鬼代言人** | 压力测试 + 隐藏假设 + 反例 | 方案的风险识别与补漏 |
| **费曼技巧** | 模拟教学 + 降维简化 + 难点定位 | 知识的彻底内化与传播 |
| **跨界联想** | 领域 A 原则 → 领域 B 隐喻 | 非线性的创新洞见 |
| **第一性原理** | 拆解至公理 + 逻辑重构 | 颠覆性的底层解决方案 |
| **行动转化** | 理论原则 → 翻译动作 → 最小动作 | 理论到执行的无缝落地 |
| **模型提取** | 经历/信息 → 核心洞见 → 命名 | 可复用的认知资产 |
| **事后复盘** | 预期 vs 现实 + 根因分析 | 失败/成功经验的复利化 |
| **未来推演** | 一阶、二阶、三阶效应分析 | 长期主义的系统思考 |
| **苏格拉底追问** | 信念陈述 → 证据探测 → 挑战假设 | 思维盲点的深度清理 |

---

## 2. 逻辑骨架 (认知内化与外化的闭环)

### **A. Chunking (组块化 - 修炼内功)**
- **位置**：大脑内部。
- **动作**：`理解 -> 练习 -> 回忆`。
- **结果**：形成可以自动调用的“心智组块”。

### **B. Zettelkasten (卡片盒 - 打造外设)**
- **位置**：外部系统 (如 Obsidian)。
- **动作**：`原子化写作 -> 建立链接 -> 索引生成`。
- **结果**：构建出能产生新思想的“第二大脑”。

### **C. 协同工作流**
`输入学习 -> 使用 Chunking (学透节点) -> 使用 Zettelkasten (存入网络) -> 建立链接 (碰撞洞见)`。

---

## 3. 避坑指南 (Brutal Truths)

- **模式不是自动机**：Prompt 提供思考路径，但“那个精彩的比喻”必须由你（或引导 AI）产出，拒绝脑力偷懒。
- **原子化是刚需**：笔记太大无法链接，思想太杂无法组块。一张卡片只说一件事。
- **链接决定生命力**：没有链接的卡片盒只是知识的坟墓，而非生长的森林。
- **拒绝“收藏家谬误”**：掌握 Prompt 模板不代表掌握思维能力。只有在实战中跑过路径，认知资产才属于你。

---

## 4. 决策树

1. **面对新领域/复杂概念？** → 深度解析 + 费曼技巧。
2. **要做关键决策/重大计划？** → 魔鬼代言人 + 未来推演。
3. **寻求创新突破？** → 第一性原理 + 跨界联想。
4. **进行复盘反思？** → 事后复盘 + 心智模型提取。

---

**原则**：工具是思维的延伸。用 Chunking 练强大脑，用 Zettelkasten 连通世界，用 Pattern 规范路径。
