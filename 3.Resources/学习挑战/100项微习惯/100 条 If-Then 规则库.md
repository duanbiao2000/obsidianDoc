## Active-Core 20（日常看板）

### 深度工作 10

1. **R01 启动 / Top1**  
   - If 起床 10 分钟内还没坐到工作位  
   - Then 坐下→打开任务清单→写今日唯一 Top1→开 25 分钟计时。

2. **R02 抗「假忙」：选一件事**  
   - If 开工 5 分钟内仍在收集/整理/找感觉  
   - Then 从清单里强制选 1 件事，加 `⭐`，今天只认这一个优先。

3. **R03 下一步不可见 → 写 1 步**  
   - If 盯着任务 > 2 分钟不知道怎么开始  
   - Then 写一句“下一步 = 动词 + 对象（≤2 分钟可启动）”，立刻执行。

4. **R04 拖延 >3 天 → 15min MVP**  
   - If 同一任务连续拖延 > 3 天  
   - Then 把交付降级成 15 分钟 MVP，今天必须完成。

5. **R05 大项目 → 30min 评审片段**  
   - If 项目大到不知道怎么拆  
   - Then 写出第一个 30 分钟可完成的“可评审片段”，先做它。

6. **R11 进入深度块 Protocol**  
   - If 打算进入深度工作  
   - Then 手机远离+仅留当前材料+设 25–50 分钟计时。

7. **R12 冲动分心 → 记一笔回工作**  
   - If 出现“刷手机/开别的 tab”冲动  
   - Then 在纸上写“我想____因为____”，然后返回当前任务。

8. **R21 卡顿 >10min → 结构化**  
   - If 同一任务原地打转 > 10 分钟  
   - Then 写 3 个小标题或画 5 节点流程，从任一节点继续。

9. **R24 一小时无产出感 → 定义本小时成果**  
   - If 过去 60 分钟“很忙但说不出产出”  
   - Then 写一条“本小时结束前必须产出的可见成果”，立即执行。

10. **R31 专注块收尾 → 2 行状态转移**  
    - If 一个专注块结束  
    - Then 写 2 行：完成了什么 / 下一步是什么，并写进任务系统。

### 数字管理 4

11. **R41 无意识解锁 → 立即锁屏**  
    - If 解锁手机 3 秒内说不出目的  
    - Then 立刻锁屏，把手机放到视线外。

12. **R42 社交 >20min → 强制中断**  
    - If 社交 App 连续使用 > 20 分钟  
    - Then 退出 App，起身走动 5–10 分钟，再决定是否继续。

13. **R44 深度块 → 全局勿扰**  
    - If 进入深度工作块（配合 R11）  
    - Then 一键开电脑+手机勿扰，仅保留电话 / P0 告警。

14. **R56 输入 >60min 无输出 → 20min 输出**  
    - If 当天信息输入（新闻/长文/视频）> 60 分钟且没有任何输出  
    - Then 用 20 分钟写一个摘要或应用片段。

### 健康 4

15. **R61 睡眠债 3 天 → 砍项目+早睡**  
    - If 连续 3 天睡眠 < 6.5 小时  
    - Then 本周砍 1 个非必要项目 + 当天提前 1 小时上床。

16. **R64 久坐 >60min → 2min 起身**  
    - If 连续坐着 > 60 分钟  
    - Then 起身 2 分钟：走动 + 颈肩拉伸。

17. **R66 15:00 前步数 <3000 → 10min 走路**  
    - If 15:00 前步数 < 3000  
    - Then 立即走 10 分钟（室内外均可）。

18. **R70 高压 ≥8/10 → 呼吸 + 下一步一句话**  
    - If 压力自评 ≥ 8/10 且持续 > 15 分钟  
    - Then 做 3 分钟呼吸 + 写下一条“我能控制的下一步”。

---

### 财务 1

19. **R81 发薪日 → 自动三分法**  
    - If 工资到账  
    - Then 按既定比例自动划转：应急金 / 投资 / 固定账单。

---

### 关系 / 声誉 1

20. **R91 承诺 = 范围 + 截止**  
    - If 对别人做任何工作承诺 / 帮忙承诺  
    - Then 同时说清“范围 + 截止时间”；不确定就说“我 X 点前给你答复”。

## 选型推理（压缩版）

1. **深度工作 10 条**  
   - 1–5 解决「启动 / 拆解 / 拖延」；  
   - 11–12+21+24+31 解决「专注 / 抗干扰 / 卡顿 / 伪工作 / 状态转移」。  
   → 这 10 条已经构成一个可 standalone 运转的「个人执行引擎」，其余规则都是对此的微调与例外处理。

2. **数字管理 4 条**  
   - 41/42 -> 控制手机这一最大注意力泄漏口；  
   - 44 -> 与深度块绑定的环境锁；  
   - 56 -> 对信息摄入施加 ROI 约束（无输出就不再加输入）。  
   → 本质上是「注意力边界条件」。

3. **健康 4 条**  
   - 61 -> 睡眠债阈值，强制与项目数量联动；  
   - 64/66 -> 久坐 & 活动量基线；  
   - 70 -> 高压即时降载 + 行为 re-focus。  
   → 这 4 条将「身体状态」显式挂到任务/项目层，而不是交给意志力。

4. **财务 1 条**  
   - 81 -> 把“先分钱给未来，再给现在”自动化，等价于植入一个长期资金流向偏好。  
   → 其它财务规则都是在出问题后才触发；81 是前置的。

5. **关系 1 条**  
   - 91 -> 把所有承诺重写成「合同形式（范围+截止）」，是职业声誉最底层的硬约束。  
   → 其他关系规则多数是 conflict handling / 礼貌增强；91 是基础契约。
