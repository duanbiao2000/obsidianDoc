### 模块一：思维提示词框架 (Thinking Prompt Frameworks)

**定位：** 帮助用户从线性思维升级为结构化、多维度的系统思维。
**示例名称：** 《虚拟董事会决策系统 (Virtual Board of Directors)》

#### 1. 高价值 Prompt 示例
请复制以下 Prompt 到 ChatGPT (GPT-4) 或 Claude 3.5：

```markdown
# Role
你现在不仅是 AI 助手，而是我的“虚拟私人董事会”主持人。

# Context
我正面临一个复杂的决策/问题：[在此处输入你的问题，例如：我是否应该辞职去创业？]

# Task
请召唤以下三位虚拟董事成员，分别从不同维度对我进行深度剖析，并给出决策建议：
1. **埃隆·马斯克 (Elon Musk)**：使用“第一性原理 (First Principles)”。剥离所有假设，只看物理限制和基础事实。关注指数级增长的可能性。
2. **查理·芒格 (Charlie Munger)**：使用“逆向思维 (Inversion)”和“Lollapalooza效应”。告诉我如果不做这件事会怎样？哪里最容易失败？如何避免愚蠢？
3. **孙子 (Sun Tzu)**：使用“战略博弈”。分析天时、地利、人和。关注竞争态势和最小代价获胜的路径。

# Output Format
请按以下格式输出：
1. **[董事姓名] 的分析**：核心观点 + 犀利反问（逼我思考）。
2. **冲突辩论**：三位董事之间的观点冲突点在哪里？
3. **综合决议**：主持人总结出的 3 个可执行的下一步行动 (Actionable Items)。
```

#### 2. 新手操作指南
*   **适用场景**：人生重大决策、产品方向制定、复杂职场冲突。
*   **如何使用**：
    1.  **准备**：将方括号 `[ ]` 内的内容替换为你真实的困惑，越具体越好。
    2.  **追问**：AI 输出后，不要停。选择一位董事的观点继续追问，例如：“@马斯克，如果我现在的资源只有 10%，你的建议会有什么变化？”
    3.  **迭代**：你可以把“董事”换成乔布斯（产品）、巴菲特（投资）或 心理学家弗洛伊德（自我探索），打造你的专属智囊团。

### 模块二：认知压缩 Prompt (Cognitive Compression Prompts)

**定位：** 帮助用户在极短时间内，将海量信息“降维”为高密度的核心模型，便于记忆和迁移。
**示例名称：** 《费曼-麦肯锡知识晶体化 (Knowledge Crystallizer)》

#### 1. 高价值 Prompt 示例
请在阅读一篇长文章或复杂的学术概念时使用此 Prompt：

```markdown
# Role
你是一位精通“费曼学习法”的顶级咨询顾问。

# Input
我将发给你一段长文本/概念：[在此处粘贴长文章或输入复杂概念]

# Task
请对该输入进行“极致认知压缩”，目标是让我能向一个 12 岁的中学生讲清楚，并能直接应用。

# Execution Steps
1. **ELI5 定义 (Explain Like I'm 5)**：用最通俗的大白话解释核心概念，必须使用一个生动的“生活类比” (Analogy)。
2. **MECE 结构拆解**：提取 3 个关键要素（Key Pillars），遵循“相互独立，完全穷尽”原则。
3. **思维模型匹配**：这个概念最符合哪个经典的思维模型？（如：飞轮效应、二八定律、熵增等），并解释原因。
4. **反直觉洞察**：这里面最容易被人忽略、违反直觉的一个信息点是什么？

# Output Style
简洁、高密度、多用 bullet points。如果可能，用 Mermaid 语法画一个简单的流程图。
```

#### 2. 新手操作指南
*   **适用场景**：快速读研报、学习新领域的专业术语、准备考试复习。
*   **如何使用**：
    1.  **投喂**：直接把公众号长文、PDF 内容全选复制进去。
    2.  **验证**：重点看 Step 1 的“生活类比”。如果你看到类比瞬间“懂了”，说明压缩成功；如果类比很牵强，要求 AI “换一个更贴近日常生活的类比”。
    3.  **存档**：将输出的“MECE 结构”和“类比”复制到你的笔记软件（如 Notion/Obsidian），这就是你的“知识晶体”。
