## 🎬 视频托管方案

### 平台选择

- **Bilibili**: [https://www.bilibili.com/](https://www.bilibili.com/)（主要）
- **YouTube**: [https://www.youtube.com/](https://www.youtube.com/)（备用）

### 视频规范

- **分辨率**: 1920x1080 (1080p)
- **时长**: 5-15 分钟
- **格式**: MP4
- **字幕**: 中文（CC）

## 📝 截图制作指南

### 工具推荐

1. **OBS Studio** - 录制屏幕
2. **LICEcap** - 制作 GIF 动图
3. **ShareX** - 截图和标注

### 制作标准

- **尺寸**: 1920x1080 或 1280x720
- **格式**: GIF（动图）或 PNG（静态）
- **标注**: 添加必要的文字说明
- **文件大小**: GIF < 5MB
